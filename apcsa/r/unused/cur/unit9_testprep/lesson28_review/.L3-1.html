Key Terms:

  • Abstraction
  • Code Reuse
  • String

Key Concepts:

  • Abstraction is the process of noticing similarities in examples and identifying the pattern that all the examples fit.
  • String objects represent sequences of characters.
  • String constants are surrounded by double-quote marks. "This could be a string constant."
