New Terms:

  • Boolean Variable
  • Boolean Method
  • this
  • The not operator (!)

Key Concepts:

  • Boolean variables can take one of two values: true or false.
  • Boolean methods return a boolean variable. These types of methods are sometimes called predicates.
  • Boolean expressions use && (and) and || (or) to precisely specify what evaluates to true and what evaluates to false.
  • There are a number of different comparison operators. The most basic is ==, used to test equality between basic data types.
  • A second type of comparison uses the .equals() method of a class. This is how Strings are compared.
  • The == operator compares identity for objects rather than value. This is why objects often have .equals() methods.

Comparisions can be tricky. If you get a question on the AP test that involves them be sure to carefully consider exactly what the code will do.
