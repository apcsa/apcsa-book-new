New Terms:

  • Source Code
  • Compiling
  • Compile-Time Error
  • Accessibility
  • Variable Type
  • Method Signature

Key Concepts:

  • Source code is compiled into object code by a compiler.
  • The compiler recognizes errors within your code that prevent compilation. These are called compile-time errors.
  • By convention, names of classes in source code start with capital letters while names of variables start with lower-case letters.
  • The accessibility of a class, variable, or method describes what other classes have access to it.
  • A method's signature (or header) is the method's name, argument types, and order of arguments.
