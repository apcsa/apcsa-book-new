New Terms:

  • Behavior
  • State
  • Class
  • Instance

Key Concepts:

  • Behavior and state are basic ways to represent real-world objects.
  • Items with similar behaviors act similarly and items that have the same states know similar things about the world.
  • Behavior and state are encoded in classes as methods and variables.
  • A class is like a set of instructions describing how to create an object.
  • Java uses these instructions to build instances of that class.
