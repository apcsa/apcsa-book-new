New Terms:

  • Java Library
  • Import
  • Callback Method

Key Concepts:

  • Java comes with a vast collection of built-in libraries that help you do useful things. These could include drawing to the screen or using specialized mathematical functions.
  • You access these libraries in your code by importing them.
  • A call-back method is a method that is never directly invoked by your code, but that is triggered by outside libraries.

Java graphics will not be covered on your AP exam. But importing libraries certainly will be!
