New Terms:

  • Pseudo-random Number Generator
  • Simulation

Key Concepts:

  • java.util.Random is a pseudo-random number generating class that allows you to add randomness to your programs.
  • Similar functionality can be found in java.lang.Math
  • One way to test the correctness of a program that includes randomness is to test it many, many times. This process is called simulation, and if it produces results in line with what you would expect (in aggregate) your code is likely working.

Randomness can be used to generate lots of different types of data in your programs. Before you take the AP test, go back to Lesson 10 and review some of them!
