New Terms:

  • abstract class
  • abstract method
  • Interface
  • static variable
  • implements

Key Concepts:

  • An abstract class can't be instantiated, but it can be extended.
  • A class with abstract methods has to be declared as abstract.
  • A subclass that provides definitions for the superclass's abstract methods can be instantiated.
  • An interface has all abstract methods and no instance variables.
  • Classes can implement an interface by providing definitions for all of its methods.
  • An interface specifies a contract for a class that implements the interface, which then can be relied upon to supply the relevant methods for users of the class.
  • Static variables are associated with classes rather than objects. All instances of a given class share that class's static variables.

All three of these keywords are very likely to show up on your AP exam. Now would be a great time to go back and review all of the material in Lesson 17!
