New Terms:

  • arrays

Key Concepts:

  • An array is a sequence of a specified number of elements, all of the same type.
  • Arrays can be initialized empty: arrayVar = new type[size];
  • Or with data already in them: int[] monthLenghts = {31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
  • Array elements are accessed by: arrayName[index number]
  • arrayVariable.length gives you the length of an array.
