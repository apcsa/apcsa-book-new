New Terms:

  • recursion
  • base case
  • recursive case

Key Concepts:

  • Recursion is when a method calls itself.
  • A recursive call essentially creates a clone of the method which goes off and does its own thing.
  • The base case is the condition in a method that does not result in a recursive call.
  • The recursive cases are any conditions that do result in recursive calls.

Recursion can be really tricky. Make sure you totally understand the concepts of recursive and base cases before you move on.
