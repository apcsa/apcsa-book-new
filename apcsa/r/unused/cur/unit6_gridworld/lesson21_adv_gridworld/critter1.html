The Critter class overrides the Actor act method, calling five methods:

  • getActors, which gathers interesting actors into an ArrayList;
  • processActors, which processes the actors in the list;
  • getMoveLocations, which gathers possible locations to move to;
  • selectMoveLocation, which chooses one of the possibilities;
  • makeMove, which moves the Critter to its new location.

By default, it gathers the neighboring actors, "eats" them (removes them from the grid), and moves to a randomly chosen empty neighboring location. Appendix C contains the Critter code.
