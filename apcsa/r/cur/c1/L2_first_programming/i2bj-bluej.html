A development environment is a tool that helps you to write, build, and organize the files and classes in your program. In this course, we will be using the BlueJ development environment to create Java programs.

This section introduces the key aspects of BlueJ you'll need to get started. Over the course of the semester we'll progressively introduce additional features.

Note: To proceed, make sure that BlueJ is installed on your computer. The first time you open up BlueJ, it will need to be configured. The Configure BlueJ page will show you how to get BlueJ running with the course.


First, start up BlueJ. The details of how you do this depend on what kind of computer and installation you are using.

BlueJ is like any other application on a computer, and starting it up should be similar to how you install/launch other applications. Your computer may have shortcuts on its desktop, or other ways to make it easy to open.

If your computer has a typical installation, you can always find BlueJ at C:\UCCP\APCSA\BlueJ\ (if you are using a Windows machine) or at /UCCP/APCSA/BlueJ (if you are using a Macintosh).
