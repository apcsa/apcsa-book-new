Now that we have an object, we can call its methods. Right-click on the gameBoar1 object on the object bench, and select void showBoard() from the list of options. A new board will be drawn in another window. 1

The list of options that shows up when you right-clicked on the gameBoar1 object are the methods of the GameBoard class (and, therefore, of objects built from that class).

The name of the method is showBoard. You'll notice that BlueJ adds more information than just the name, however.

  • The word void at the front indicates what type of thing the method will return after it has been run. This method returns nothing, which in Java is described as void.
  • The two parentheses at the end are indicating what parameters the method needs to receive. There is nothing between the parentheses, so this method takes no parameters.

Selecting a method calls the method on the object. Try selecting void hideBoard(). Does it do what you expect?
