We have discussed the distinction between classes and objects in previous activities. In BlueJ, we build, or construct, an object from a class. Here's how we do it.

Right-click on the GameBoard class. From the list options, choose Compile. By choosing compile, you have asked the computer to turn the source code into object code What do you see on the screen?

   Here is a slide show of these steps if you prefer an illustration: 1

Right-click on the GameBoard class. From the list options, choose new GameBoard(). You will be asked for a "name of the instance"; clicking OK will do for now. What do you see on the screen?

You should see a rectangle at the bottom of the screen called gameBoar1: GameBoard. Congratulations, you've just made your first object in BlueJ!

gameBoar1 may seem like a strange name for a GameBoard. What happened to the 'd'? By default, BlueJ uses the first eight characters of the class name followed by a number, so the 'd' got cut off. If you named it yourself, you could name it gameBoard1 or any other legal Java name. However, for now, gameBoar1 will work just fine.
