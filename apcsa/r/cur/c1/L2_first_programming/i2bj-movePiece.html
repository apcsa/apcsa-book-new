Right-click one of the objects of the class GamePiece that you created previously and select moveRight(int). The int between parentheses says that the moveRight method takes a parameter of type int (int is short for integer, a number without a decimal point. 3, 872, and -4 are integers. 3.1415 and 8.1 are not).

A dialog box will allow you to type in something. Type in 3 and click OK. Notice what happened to the game piece. Now try typing in 50 and observe the outcome.

A call to moveRight() doesn't take a parameter. It moves the piece to the right for a fixed distance (one square). In contrast, the moveRight(int) method lets you specify a number telling the method how far to move the piece.
