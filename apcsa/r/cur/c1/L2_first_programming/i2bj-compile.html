The instructions on this page are illustrated here.1

Make some changes to the GameBoard class. (A simple change is just to click on a blank space and type another blank space.) Now look at the rectangle corresponding to the GameBoard class. How did it change?

BlueJ indicates that the source code of a class has been changed by drawing stripes on the rectangle for that class. If we want the changes to take actual effect, we need to compile the class, which will be explained next. Now, just click on the Compile button. Check that the stripes have disappeared.
