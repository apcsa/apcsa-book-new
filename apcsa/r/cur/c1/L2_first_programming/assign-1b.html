A possibly more confusing assignment is the statement in refill that increases myCandyCount: 

public void refill (int amount) {
   myCandyCount = myCandyCount + amount;
}

In algebra, a statement like myCandyCount = myCandyCount + amount would make no sense, at least if amount weren't zero. But in Java, here's what happens:

  1. The value in the box named myCandyCount is added to the value in the box named amount.

  2. The result of adding in Step 1 replaces the value in myCandyCount.

The statements in a method are referred to as the method body. The bodies of the various CandyJar methods each consist of only one assignment statement. We will soon, however, see examples of longer methods.

To avoid confusion, we usually read the equals sign in an assignment as "gets", for example, "myCandyCount gets myCandyCount plus amount".

Another way you could say this would be "make myCandyCount equal to myCandyCount plus amount".
