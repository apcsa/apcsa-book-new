Select the actions that produce an error.
Changing public to private in the header of refill, then calling refill.
Correct. Declaring a variable or method private limits access to it.
Switching two of the methods.
It doesn't matter to Java what order the methods are in.
Switching public and void in the refill header.
Correct. The keyword public must precede void. More generally, if a method is returning a value, public must precede the return type of the method.
Moving the instance variable declaration to the end of the class, after all the method declarations.
Method and instance variable declarations may appear in any order.
