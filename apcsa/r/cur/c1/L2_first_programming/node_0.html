Let's examine the process in a little more detail. When you selected the new GameBoard()option from the menu, you are actually calling a special method in the GameBoard class referred to as a constructor. 1 This call makes a new instance of the GameBoard class with the default name of "gameBoar1", which is why it is labeled as it is.
