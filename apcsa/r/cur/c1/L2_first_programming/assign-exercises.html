Let's practice using assignment statements.


Open BlueJ, and open the lesson2/CandyJar project


Find the method doubleCandyAmount in the CandyJar class.

In the colored box, write code for the doubleCandyAmount in the CandyJar class so that when called, the current amount of candy in the candy jar is doubled.

You can only write in the colored boxes; the rest of the file can't be changed. Once you think you've got it, click the "Test" button at the top of the editor window. BlueJ will tell you if it thinks you got it right!
