When BlueJ first opens, you will see a window with a few buttons on the left and a large, mostly blank area. Let's open the GameBoard project, a simple Java version of the demonstration you saw earlier.

There are special BlueJ steps in the project that are linked to the BlueJ application. When you see the box below, it means you are on a special BlueJ step.

Open the GameBoard project

Start BlueJ, if you don't already have it running.

Open the current BlueJ project: To open the current BlueJ project, which for this step is the GameBoard project, click Tools on the menu bar and select Open Current BlueJ Project from the drop-down menu. The GameBoard project should automatically load, because you are at a special BlueJ page. Try it now.

Did you have trouble opening the project? Here's the help page describing alternative ways of opening a bluej project.

A project is simply a collection of files containing the Java statements that makes up the program. In BlueJ, every program is organized into its own project.
