The way to change the value of a variable is to assign it another value, by using an assignment statement.

The assignment statement starts with the name of a variable (the one whose value we want to change), continues with an equals sign, and finishes with an expression that specifies the replacement value. Here is a straightforward example, the initialization of myCandyCount in the constructor:







public CandyJar (int startingCount) {
   myCandyCount = startingCount; 
}

This says to copy what's in the box named startingCount into the box named myCandyCount, replacing whatever was there before.
