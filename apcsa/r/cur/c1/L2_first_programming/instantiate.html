To use an object, we must build it by calling the class constructor. This produces an instance of the class. If the constructor has parameters, we must supply values of the appropriate types for those parameters, in the same order as they appear in the constructor header.

These values we supply are called arguments.


Open the BlueJ project for this page.

(If you have forgotten how to do this, go to the "Tools" menu in BlueJ, and select the "Open Current BlueJ Project" menu item. The current project is in the lesson2 directory, and is named CandyJar, of course.)


Compile the CandyJar class.

Right-click on the CandyJar class to instantiate a CandyJar object that initially contains 5 pieces of candy.

Accept BlueJ's suggestion of the name; candyJar1 for this object.
