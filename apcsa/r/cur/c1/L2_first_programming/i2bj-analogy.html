  • You supply the text of a program. Programmers call this source code.
  • The Java compiler makes sure it is legal Java; if not, it complains, and you have to fix what's wrong. The Java compiler is very particular when looking over your program. Even the smallest error will cause it to stop and complain.
  • If your programming code contains no errors, the compiler translates it into something that can be executed. Programmers call this object code.
