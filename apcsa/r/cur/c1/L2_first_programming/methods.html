The CandyJar class has four methods.

The eat method implements the eating behavior we talked about earlier. Eating some of the candy means reducing the value of the myCandyCount instance variable. 

public void eat (int amount) {
   myCandyCount = myCandyCount - amount; 
}

The second method is refill. Refilling a CandyJar increases the value of the myCandyCount instance variable.

public void refill (int amount) {
   myCandyCount = myCandyCount + amount; 
}

Method names follow the same conventions as variable names for capitalization: they start with a lower case letter and additional words are capitalized.

But, they don't generally start with the word "my", because that isn't a natural way to talk about behaviors.
