Suppose that myCandyCount contains the value 7, and the eat method is called with an argument of 3. Which of the following are variables that are accessed as a result of that call?

eat
This isn't a variable! It is a method.
myCandyCount
Right: when eat is called, myCandyCount is both used to calculate the new value as well as assigned that new value.
amount
Right: when eat is called, the amount variable is used to calculate the new amount (myCandyCount).
startingCount
startingCount isn't a instance variable or parameter used by the eat method!
