Naming the ingredients

The average use is determined by dividing the sum of values passed to useUp by the number of calls to useUp.   We thus need two more instance variables to be able to compute the average:

  1. the number of times useUp is called, and
  2. the sum of the arguments supplied in each call to useUp

What are good names for these variables?

1. useUpCount will accumulate how many times useUp is called in total, so I think a variable that serves as a counter should have the term 'count' in it. 2. allThingsUsed is a good name because it will keep count of every argument that is supplied in a call to useUp.
