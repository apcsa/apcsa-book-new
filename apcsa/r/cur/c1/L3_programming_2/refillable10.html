Alternative names for Refillable methods

We could have chosen different method names for the Refillable. Consider the name useUp, the name of the method that reduces the amount (of something) in the refillable object.

For each of the following, briefly explain how it compares with the name "useUp".

  • decrease
  • consume
  • reduce
- decrease: I think the word decrease doesn't indicate that the amount of something is being used up. For example, decrease is perfectly fine for the recording scores, but it doesn't really make sense if decrease is used to represent the drink method. - consume: I think it demonstrates useUp well, but opposite of 'decrease', it might not make sense if we are talking about the cokeCooler class. If consume is used in the cokeCooler class, does that mean the cokeCooler drank the coke? (!!!) - reduce: I think it's the same with 'decrease'.
