The String substring method returns a different String that is a substring of the original.

The String class has two different versions of the substring method. Since both of the methods are named substring, how does Java know which one to use? Java is able to distinguish between the two methods based upon the number of arguments passed to the methods. For example,

s.substring(5);

is a different method call than

s.substring(5,2);

because the first method has only one argument and the second method has two arguments.
