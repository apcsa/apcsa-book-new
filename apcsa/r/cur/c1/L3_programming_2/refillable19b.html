The average is computed by dividing the total use amount by the number of calls to useUp. This is done using the slash ("/") operator. The statement:

var1 / var2

where var1 and var2 are int variables, is executed by dividing the value in var1 by the value in var2 and then discarding the fractional part.

For example, the value of    7/3   is  2.

The reason you discard the fractional part is both variables are of type int, and an int can only be an integer, not a fraction or number with a decimal.

Java does have numeric types that can have decimals. We'll look into those later.
