The plus sign, "+", has a different meaning when used with String objects than it does with int values.

Applied to two strings, "+" means concatenation. What results is a string whose value is that of the two original strings, glued together.

For instance, the statement:

"Hillary" + "Clinton"

will result in the string 

"HillaryClinton"

Note that there isn't any space between the words, because neither of the strings that were glued together had a space.
