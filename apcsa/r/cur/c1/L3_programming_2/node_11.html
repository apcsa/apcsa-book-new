We've seen, in the Name class, the access of methods via the dot operator. For example, if rtest is a variable of type Refillable, then 

   rtest.useUp (10);

is a call to the Refillable method named useUp.

We include enough information in the output to simplify the checking of the effect of each method call.

The line 

   Refillable rtest = new Refillable (100);

declares and initializes a variable named rtest of type Refillable. We'll explore this line in detail shortly.
