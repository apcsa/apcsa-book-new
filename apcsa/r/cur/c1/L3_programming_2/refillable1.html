You are ready to design your own class to model something that's similar to a candy jar. Here's an example.

Google (the company that developed the Internet search program) provides free soft drinks for its programmers. In particular, they have a refrigerator that contains only cans of Coca Cola™. In the next few steps, you'll design a CokeCooler class that provides the following methods:

  • a constructor that takes a positive integer as an argument (the number of Cokes initially in the refrigerator);
  • a drink method whose single integer argument says how many Cokes are being removed from the refrigerator;
  • a refill method whose single integer argument says how many Cokes are being added to the refrigerator;
  • a cokeCount method that returns the number of Cokes in the refrigerator.
