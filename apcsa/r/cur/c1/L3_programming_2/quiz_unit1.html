Question A

Given below is a class that represents amounts of money in dollars and cents.

public class Money {

   private int myDollars;
   private int myCents;
   
   public Money (int dollars, int cents) {
      myDollars = dollars;
      myCents = cents;
      fix ( );
   }
   
   private void fix ( ) {
      int totalCents;
      totalCents = myDollars * 100 + myCents;
      myDollars = totalCents / 100;
      myCents = totalCents - (myDollars * 100);
   }
   
   public String toString ( ) {
      String centString;
      centString = "" + (100 + myCents);
      return "" + myDollars + "." + centString.substring (1);
   }
  1. Identify each of the following as either an instance variable, a local variable, a parameter, an argument, or an object. (There may be more than one answer.) 
    myDollars
dollars
totalCents
  2. Give an example of values one could supply to the constructor that would result in myDollars and myCents containing different values before and after fix is called.
  3. Describe the effect of interchanging the assignments to myDollars and myCents in fix, by describing as completely as possible the values of dollars andcents for which the modified fix would assign different values to myDollars and myCents.
  4. What values are assigned to centString when myCents contains:
    • 0
    • 5
    • 10
    • 99
