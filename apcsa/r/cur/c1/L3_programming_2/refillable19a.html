Let's take a closer look
at the new useUp method. 

public void useUp (int amount) {
   myAmount = myAmount - amount;
   myUseUpCallCount = myUseUpCallCount + 1; // NEW
   myTotalUse = myTotalUse + amount; // NEW
}

Note that a method may contain more than one statement. In fact, most methods you write will do so.

When you have a method with more than one statement:

Each statement ends with a semicolon.

The statements in the method are executed in order, from top to bottom.
