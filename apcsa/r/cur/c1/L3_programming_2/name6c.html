BlueJ's Code Pad lets us try out Java features without writing entire classes.

Explore the + operator.

Open BlueJ now. It doesn't matter what BlueJ project your have open.

Select Show Code Pad from the View menu. A new pane will appear in the lower-right part of the BlueJ design view. This is the BlueJ "code pad". You can type simple Java statements here and immediately see what they return.

Next, type the expressions below and look at what is returned.

12 + 23
	  	
"12" + "23"

As you can see, concatenation of strings and adding of numbers are quite different, even though they both use the operator +.
