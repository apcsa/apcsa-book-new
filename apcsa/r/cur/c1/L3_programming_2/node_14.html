Either a primitive variable or a reference variable may be initialized to the value of another variable of the same type, for example: 

   int myAmount;
   myAmount = 5;

   int initialAmount = 5;
   int myAmount = initialAmount;
   
   String namePart = "HILLARY";
   
   String temp = namePart.toLowerCase();

A primitive variable may also be initialized to a constant of the appropriate value, e.g. 

   myAmount = 0;

The analogous way to initialize a reference variable is more complicated.
