Suppose we were to insert the multiplication by 1.0 into the expression without parentheses. Which of the following work as desired? 
public double averageUse ( ) { 
 return 1.0 * myTotalUse / myUseUpCallCount;
}
Correct! 
public double averageUse ( ) { 
 return myTotalUse * 1.0 / myUseUpCallCount;
}
Correct! 
public double averageUse ( ) {
 return myTotalUse / 1.0 * myUseUpCallCount;
}
That returns myTotalUse times myUseUpCallCount! 
public double averageUse ( ) {
 return myTotalUse / myUseUpCallCount * 1.0;
 }
That multiplies the truncated quotient by 1.0.
