The first of the two substring methods has only one argument, an integer which represents the starting index of the substring. The substring that is returned by this method includes all the characters starting with the index going all the way to the end of the original string. For example,

   String s1 = "teddy bear";
   String s2 = s1.substring(6);

would result in s2 being set equal to the substring "bear".

The second substring method has two arguments, an integer representing the starting index and an integer representing the ending index, well almost. You have to subtract one from the second argument to get the ending index. This is a source of confusion for many beginning students. For example,

 
   String s1 = "teddy bear";
   String s2 = s1.substring(0,5);

would result in s2 being set to the substring "teddy" (the substring beginning at index 0 and ending at index 4).
