As a first step of writing the averageUse method, think about how other parts of the program will use it.

Of the following choices, which is the best return type for the averageUse method?

void
No. Remember, the "type" of method is the type of the variable that it returns. void types don't return anything, but averageUse needs to return the average amount that was used in each call to useUp
int
Yes, the "type" of method is the type of the variable that it returns, and averageUse is going to return a number. You may have noticed that int can't return a number with a decimal point (such as 3.4). That is correct! And, that makes int not the perfect choice. Later in the course, we'll explore types in Java that do allow decimal points.
Refillable
No! Remember, the "type" of method is the type of the variable that it returns. Refillable is a class. This method isn't going to return an entire Refillable instance!
