The substring method provides a way to take a string apart. Another method named indexOf allows us to search a string. The indexOf method takes a String object—we'll call it strToSearchFor—as an argument, and returns the index in this string of the start of the first occurrence of strToSearchFor. If the search fails, indexOf returns –1.

Here are some examples. They all assume that String s is "december".

callreturn value
s.indexOf ("ember"); 3
s.indexOf ("deb"); –1
s.indexOf ("e"); 1
s.indexOf (""); 0
s.indexOf ("erf"); –1
