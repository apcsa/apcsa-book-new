Each variable in Java must have an associated type, specified in a declaration, for example: 

   int count;
   double pi;
   String myFirst;

Some types are primitive types, for example, int and double. String, CandyJar, Refillable, and Name, on the other hand, are called reference types because variables of these classes refer to objects.

There are only eight primitive data types. The Advanced Placement CS syllabus requires understanding only of int, double, and boolean (to be covered in the next lesson). Thus most of the variable types you encounter will be reference types.
