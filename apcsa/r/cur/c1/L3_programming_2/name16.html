The caseFixed method has some other interesting features.

  • It's a private method, intended solely for use by the Name constructor. It's not accessible from outside Name.
  • It uses a local variable named temp. Since temp is declared in caseFixed, it can be used only within caseFixed.
  • It calls several String methods. We have seen how to call methods of objects in the BlueJ workbench, but this is our introduction to calling methods inside the top-level methods. If objectName is the name of an object and methodName is the name of a method in the corresponding class, then the expression 
    objectName . methodName (parameters)
    calls the named method.
