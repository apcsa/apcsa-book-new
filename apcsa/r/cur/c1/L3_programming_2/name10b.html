In this page, you'll explore the caseFixed method by building a instance of the Name class.

Open BlueJ, and open this step's project.

(Remember, from within BlueJ, go to the Tools menu and select the Open Current BlueJ Project. This should open the Names project in the lesson3 directory.

  1. Make an instance of the Name class, using the object bench. Remember, "right-click" on the Name class, and, if it is compiled, select "New Name"..
  2. The constructor requires three parameters. Supply the strings
    • "hillARY"
    • "RodhaM"
    • "CLINTON"
    as arguments to the constructor. You will need to include the quotation marks when you enter these in the constructor dialog box. Note the weird capitalization!
  3. Then call the fullName method on your new instance, and see how it has stored those names. Remember, you call a method on your object by "right-clicking" on the object in the object bench.
