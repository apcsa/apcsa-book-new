There are three steps to initializing a reference variable from scratch:

  1. declaring the reference variable;
  2. using the new operator to build an object and create a reference to the object; and
  3. storing the reference in the variable.

Here's an example. The statement 

   Refillable rtest = new Refillable (100);

first declares the variable rtest as something that can contain a reference to a Refillable object. Then the new operator builds—allocates memory for—a Refillable object, and calls the Refillable constructor to initialize the instance variables. The new operator returns a reference to the new object, which is then stored in the rtest variable.

The variable declaration (step 1) and the initialization (steps 2 and 3) may be done separately, for example, 

   Refillable rtest;
   rtest = new Refillable (100);

The use of the new operator is handled automatically by the BlueJ system when we control-click on the class. Explicit use of new is required when we initialize an object in program code.
