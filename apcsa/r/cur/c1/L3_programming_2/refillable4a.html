Design a Score class that keeps track of how a player is doing in a game. (You don't need to test this class.)

Here are its methods:

  • a constructor that takes a positive integer as argument (the number of points that the player has earned already);
  • an increase method whose single integer argument says how much to increase the player's score;
  • a decrease method whose single integer argument says how much to decrease the player's score;
  • a score method that returns the player's current score.
