Let's examine the Name constructor more closely: 

public Name (String first, String middle, String last) {
   myFirst = caseFixed(first);
   myMiddle = caseFixed(middle);
   myLast = caseFixed(last);
}

It takes three parameters, and assigns them to each of the instance variables. However, it is using the caseFixed method to transform each of the parameters prior to assigning them to the instance variables. That is, it passes each parameter into caseFixed, and then assigns the instance variable to caseFixed's return value.

As you might expect from the name, the caseFixed method is making sure that the capitalization of each parameter is correct: that they start with a capital letter and contain only lower-case letters thereafter. We'll look more closely at it in a bit.
