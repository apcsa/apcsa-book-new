A straightforward translation of the fixFirst, fixMiddle, and fixLast methods into a single fixCase method yields the following. 

   public Name (String first, String middle, String last) {
      myFirst = first;
      myMiddle = middle;
      myLast = last;
      fixCase (myFirst);
      fixCase (myMiddle);
      fixCase (myLast);
   }

   private void fixCase (String namePart) {
      namePart = namePart.toLowerCase().trim();
      namePart = namePart.substring(0,1).toUpperCase() + namePart.substring(1);
   }
