Choosing a good name (4)

Write the method header (signature) for the increase method.   Remember that the header consists of four parts: access, type, name, and parameters.   The access for this method is public.   After you write the header, explain your choice for the type.   Also, briefly explain your choice for the names of the method and the parameter.

public void increase(int amount) The type should be void because this method does not return an answer. It just changes something. increase seems like a good name for the method because it increases the score. amount is a good name for the parameter because it is the amount by which the score increases.
