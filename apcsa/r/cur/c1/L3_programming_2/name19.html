Recall that "" (two consecutive double-quote characters) is the empty string. In Code Pad, declare a String variable named s and initialize it to the empty string as follows: 

String s = "";

Now try the following method calls in Code Pad to see what String is returned. (Don't put a semicolon at the end of these.) The string_methods link to the right provides a useful review of the commonly used String methods if needed. 

s.toLowerCase()
s.trim()
s.toUpperCase()
s.length()
