The Name class use Java Strings. The String class is built-in, a class supplied in the Java language. We will experiment with String objects, then return to the Name class.

String objects have some useful properties:

  • A String object represents a sequence of characters—basically, things you can type at the keyboard (including the space character and punctuation)
  • A String constant is surrounded by double-quote marks, e.g. "java".
  • The characters in a String are numbered: the first character (the "j" in "java") has number 0, the next (the "a" in "java") has number 1, and so on. This number is sometimes called the index of the character in the String object.
