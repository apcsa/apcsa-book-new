Bits are arranged in a different way to store double values. Each double value is stored in 64 bits, some of which represent the exponent, and some the coefficient. This is similar to scientific notation, used to represent values in biology, physics, and chemistry.

The fractional part of the coefficient is expressed in terms of fractional powers of 2. For example, the decimal value .5 is represented as 1/2, and .65625 is represented as the sum 1/2 + 1/8 + 1/32 = .5 + .125 + .03125.

Problems arise when the fraction is not exactly representable by a sum of fractional powers of 2. The fraction must then be approximated by the closest value that is represented with such a sum. An entire field of computer science, called numerical analysis, focuses on managing the inexact nature of computation involving double values.
