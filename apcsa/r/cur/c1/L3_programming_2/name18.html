No, substring will return a portion of the string, from the starting index 3 to one less than the ending index 6.

No. Remember, strings start at index 0, not index 1. The "m" is at index 2 in the string.

No, substring subtracts one from the second argument so the substring(3,6) would be the string from index 3 to index 5 inclusive.

Right! The "p" is at index 3 and the"t" is at index 5 and "put" is the substring between index 3 and 5 inclusive.
