The String class provides many methods. Documentation for the String class and for the rest of Java is available online. Java is a huge programming language, with many thousands of built-in methods.

Open a browser window to http://java.sun.com/j2se/1.5.0/docs/api/.

You have opened the "Java API". "API" stands for the term "application programming interface", which is a definition you don't need to remember. An API is a list of built-in methods, and, in Java's case, also has descriptions of classes, tutorials on how to use them, and so forth.

Let's look further:

Scroll the left menu down and select String; a page explaining the String class appears.

Scroll down to the section named "Method Detail", and then look for the methods toLowerCase, trim, and substring that are used in the Name methods.

Can you make any sense of the Java API? You will after some more time in this course!
