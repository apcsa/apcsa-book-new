Comparing two program segments

Describe all values of temp for which the following code produces the wrong initialization for comfortLevel. 

if (temp > 80) {
 comfortLevel = "too hot";
}
if (temp < 50) {
 comfortLevel = "too cold";
} else {
 comfortLevel = "comfortable";
}
For all values above 80, it produces the wrong initialization because after the comfortLevel is set to 'too hot', it continues to the next if statement. Then the comfortLevel gets set to 'comfortable'.
Every value of temp gets classified as comfortable, so any values of temp that are less than 50 or greater than 80 gives the wrong result.
There are three cases. If temp is less than 50, the code works. If temp is between 50 and 80 (inclusive), the code works. If temp is greater than 80, it doesn't work since comfortLevel gets set to "too hot", but then gets reset to the wrong value.
