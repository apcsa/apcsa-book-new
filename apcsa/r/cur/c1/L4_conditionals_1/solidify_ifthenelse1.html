Consider the Refillable class's useUp method below. 

 void useUp (int amount) {
 if (myAmount < amount) {
 System.out.println ("not enough left");
 } else {
 myAmount = myAmount - amount;
 }
 }

Suppose that myAmount contains the value 14, and useUp is called with the argument 4. Of the following statements, which is the first to be executed?

System.out.println ("not enough left");
No. Because myAmount is greater than amount, the test for the if statement will evaluate to false, and the System.out.println statement will be skipped.
myAmount = myAmount - amount;
Right. Because myAmount is greater than amount, the test for the if statement will evaluate to false, and the else statement will be executed.
Neither of those two statements will be executed.
No, one of the above statements will get executed.
