Java also provides a facility for executing one section of code if the result of a comparison is true, and another section of code if the result of the comparison is false. It's called an if...else statement. Here is its format.

if (comparison) {
   statements for a true result 
} else {
   statements for a false result 
}

It's executed as follows.

  1. The comparison is evaluated.
  2. If it's true, the statements for a true result are executed and the statements for a false result are skipped.
  3. If it's false, the statements for a true result are skipped and the statements for a false result are executed.
