  3. Provide the code for a Weight class that represents a weight expressed in pounds and ounces. Your class should provide the following public operations:

    • a constructor with one int argument, a nonnegative number of ounces that's between 0 and 15;

    • a constructor with two int arguments, a nonnegative number of pounds and a value for ounces that's between 0 and 15;

    • a boolean isHeavier method with one Weight argument that returns true if this Weight object is heavier than the argument and false otherwise;

    • a multiple method that, given a positive int argument, returns a new Weight object that results from multiplying the integer by the represented weight (e.g. multiplying 1 pound 9 ounces by 3 gives 4 pounds 11 ounces;

    • a print method that prints this Weight object as "n pounds m ounces", where n is a nonnegative integer and m is an integer between 0 and 15, inclusive.

    You may define additional private methods. The constructors are the only methods that should change the internal information in the Weight object to which they are applied.

    There are 16 ounces in a pound. Your code need not do any error checking.
