We started this lesson by observing a deficiency in the programming done thus far. Namely, there was no way to selectively evaluate statements in a program. Here's an example of what we want to do: 

public Refillable (int amount) {
   // check that amount is at least 0
   // if not, print an error message
   // otherwise initialize instance variables
}

Java's conditional statement solves this problem. Its general format is as follows. 

if (yes-or-no question) {
   sequence of statements to execute if the answer is yes
} else {
   sequence of statements to execute if the answer is no
}
Exactly one of the statement sequences is executed, depending on the answer to the yes-or-no question; the other sequence is skipped.

A special case is when there are no statements to be executed if the answer is no: 

if (yes-or-no question) {
   sequence of statements to execute if the answer is yes
}

We used conditional statements to make sure that the arguments to the Refillable methods all are reasonable.
