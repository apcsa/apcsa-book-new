Here's a solution to the leap year counting exercise: 

    if (year % 400 == 0) {
        leapYearCount = 1;
    } else if (year % 100 == 0) {
        leapYearCount = 0;
    } else if (year % 4 == 0) {
        leapYearCount = 1;
    } else {
        leapYearCount = 0;
    }

The code contains four cases, exactly one of which is executed for any particular value of year. Applying testing principle 1, we must test this code with at least one year value per case, so at least four values of year are needed for testing:

Complete the LeapYearTester class in lesson4/LeapYearTester, and run it in BlueJ. Your tests should include the following.

  • a year that's divisible by 400;
  • a year that's divisible by 100 but not by 400;
  • a year that's divisible by 4 but not by 100;
  • a year that's not divisible by 4.
