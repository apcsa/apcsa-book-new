Here are examples of path counting in various program segments:

  • Now consider two consecutive if statements: 
        if ( comparison/boolean value ) {
    }
    if ( comparison/boolean value ) {
    }
    There are two possibilities for each test, success or failure. Thus there are four paths through the two statements, corresponding to the four possibilities:
    • 1st if : true
      2nd if : true
    • 1st if : true
      2nd if : false
    • 1st if : false
      2nd if : true
    • 1st if : false
      2nd if : false
