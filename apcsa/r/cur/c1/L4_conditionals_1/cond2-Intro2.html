A conditional statement (also called an "if" statement) lets us decide to skip some statements in a program. It asks a question—the "condition". If the answer to the question is yes, the body of the if statement is executed; if the answer is no, then the body of the if statement is skipped. 

   if (yes-or-no question) {
      statements for the answer yes;
      these are skipped if the answer is no;
   }
