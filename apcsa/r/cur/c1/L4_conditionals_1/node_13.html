There are several kinds of editing styles for conditional statements, shown below. The one we use appears on the left. Your instructor may prefer one of the other styles. Above all, you should be consistent. 

if (myAmount < amount) {
   System.out.println ( ... );
} else {
   myAmount = myAmount - amount;
}
