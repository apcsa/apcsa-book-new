Objects in programming break too. One way this happens is because of bugs, internal flaws in a program. (We'll revisit bugs and how to find and get rid of them in a little while.)

Sometimes, however, the breakage arises from misuse of the object—for instance, using any convenient round object as a bowling ball, even if that round object had a different original purpose.
