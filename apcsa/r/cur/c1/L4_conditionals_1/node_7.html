A simple yes-or-no question to ask is a comparison. The comparison compares the values of two variables, or a variable and a constant. It uses one of the comparison operators:

operator meaning operator meaning
> "is greater than" >= "is greater than or equal to"
< "is less than" <= "is less than or equal to"
== "is equal to"
(Note, this is two equal signs.) 		!= "is not equal to"

Here's an example. Suppose we want to refill a Refillable object with 20 only if the current amount is 0. The code is: 

if (currentAmount ( ) == 0) {
   refill (20);
}

You could also write this code like this: 

if (myUseUpCallCount == 0) {
   refill (20);
}

Will these pieces of code work the same?
