Consider the Refillable method below. 

void useUp (int amount) {
   if (myAmount < amount) {
      System.out.println ("not enough left");
      amount = 0;
   }
   myAmount = myAmount - amount;
}

  1. Suppose that myAmount contains the value 14 at the start of useUp, and amount contains 4. What is the next statement to be executed?

  2. Suppose that myAmount contains the value 14 at the start of useUp, and amount contains 24. What is the next statement to be executed?
