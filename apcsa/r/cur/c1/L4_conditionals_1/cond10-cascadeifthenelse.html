The else in an if...else may be followed by another if or if...else. An example is the program segment below, which initializes the variable comfortLevel depending on the Fahrenheit temperature in the variable temp.

   if (temp > 80) {
      comfortLevel = "too hot";
   } else if (temp < 50) {
      comfortLevel = "too cold";
   } else {
      comfortLevel = "comfortable";
   }

Note that in the final else, we don't have to retest to make sure that temp is between 50 and 80.

We know that since the only way to reach the final else is for the first two comparisons to evaluate to false.
