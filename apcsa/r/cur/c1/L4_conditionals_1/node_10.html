We have been careful to always use curly brackets—"}" and "{"—with the bodies of the if and else statements: 

   if (myAmount < amount) {
      System.out.println ("not enough left");
   } else {
      myAmount = myAmount - amount;
   }

As it happens, the curly brackets are not necessary if the body contains only one statement: 

   if (myAmount < amount)
      System.out.println ("not enough left");
   else
      myAmount = myAmount - amount;
