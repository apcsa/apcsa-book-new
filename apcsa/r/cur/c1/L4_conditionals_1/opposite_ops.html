Sometimes it is useful or easier to think about the opposite of what you want to compare in an if statement. Try this out.

Suppose x and y are integer variables. Which of the following represents the opposite of the expression x < y ? That is, which of the following statements will be true whenever x < y is false, and false whenever x < y is true?

x == y (there are two equals signs here)
No, this is the opposite of x != y .
x <= y
No, x <= y will return false only when x is greater than y, and true otherwise. The only values of x and y for which this will return the opposite of x < y is when x is equal to y.
x > y
No, although very close! x > y and x <= y will return the same values when x is equal to y.
x >= y
Right!
There is no statement that is the opposite of x < y .
Actually, one of the above statements is the opposite of x < y
