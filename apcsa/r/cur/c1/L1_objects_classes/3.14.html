We'll now work with a simulation of a program to draw a figure on the screen that represents a 4x4 game board on which is placed one game piece. The positions on the board are pairs of numbers: a row number (between 1 and 4) and a column number (also between 1 and 4). A piece at position (1, 1) is at the upper left corner of the board; the position (3,4) is where the robot Float will land in the picture above.

Our game board class is organized as follows. The internal state is the position of the piece (a row number and a column number) and the color to make the board. There are several methods:

  • A constructor.
  • drawBoard ( ) draws the whole board.
  • drawHorizontalLine (int) draws a horizontal line at a given position on the screen.
  • drawVerticalLine (int) draws a vertical line at a given position on the screen.
  • drawPiece ( ) draws the piece.
  • move (int, int) moves the piece to the given position and changes its state accordingly.
  • erase ( ) erases the board.
