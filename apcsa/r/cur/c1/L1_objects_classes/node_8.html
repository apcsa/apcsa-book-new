The tick behavior included a comparison to check if the clock time was the same as the alarm time. This use of tests to determine how the behavior should proceed is quite common.

Another useful distinction among behaviors is whether they are activated externally (e.g. by the person using the clock) or internally (by some other clock behavior). Externally initiated behaviors we've seen so far are

  • set the alarm time to _____
  • set the clock time to _____
  • disable the alarm sound
  • enable the alarm sound
  • shut the alarm off

The following behaviors are activated from inside another clock behavior.

  • tick
  • sound the alarm
  • compare the clock time and the alarm time
