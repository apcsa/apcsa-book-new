Data Representation

How would you represent a date in this century (say, December 25, 2008)?

I'd have a string for the month and numbers for the day of the month and the year.
For a date in this century, I think I would use a number, counting from the number of days since the start of the 21st century.
I would have a string that looks like a standard date, for example 9/10/2010.
I would use three numbers, one for the month, one for the year, and the other for the date.
