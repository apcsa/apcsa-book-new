This set of activities reveals more details about how state information is represented in programming objects. In particular, a state variable may consist of ...

Numbers
Sequences of characters (referred to as strings and surrounded with double-quote characters when programming)
Yes-or-no, or true-or-false, information. This is called Boolean information.
Combinations of all three, by using a variable to reference other objects that have their own state variables (which may include more objects, and so on, and so on).
