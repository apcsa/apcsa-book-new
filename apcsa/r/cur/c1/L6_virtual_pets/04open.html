Let's use BlueJ to open Cam's first Virtual Pet write-up. We'll review what she wrote.

Open this page's project in BlueJ.

(The project is /lesson6/cam-v1)

You should see two classes named VirtualPet and VirtualPetFace. Each object of the class VirtualPet represents a virtual pet toy. VirtualPet has an instance variable of type VirtualPetFace used to display the image related to the virtual pet's mood.

Make a VirtualPet object now, using the object bench in the usual way. You can name it anything you like.

A window will open, containing an image representing the mood of the your Virtual Pet you just made. Cam has programmed the Virtual Pet to show a "normal" mood at the start.

The virtual pet has behaviors that you can invoke in the usual way. You'll see the methods "feed", "exercise", and "sleep".

Invoke all three methods on your VirtualPet object. See what happens!
