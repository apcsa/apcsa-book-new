First, a word about the Virtual pet images that we've given you. You can see all of these images inside your BlueJ project folder, but not using the BlueJ program itself. Rather, use your computer's operating system to look at the files directly. They are in the directory 

 Projects/lesson6/cam-v2/pet_images

Copy the folder cam-v2 to a folder titled VPProject. This is the project that you can start working on.

Another easy way to copy a project is from within the BlueJ program, first opening up the project you want to copy and then using the Save As menu to save it as another project. The images will get copied as well.

We've provided many, many images, but it is likely that you are going to want more as you program your own ideas. While our artist made great images, you can add files yourself, either from things you get on the internet or create yourself. You'll need to follow the same naming scheme that we've used with our images, based on the mood that you want to show.

For instance, if you want a mood titled broken-toe, you'll need to get an image into that directory titled pet_broken-toe_1.jpg. (Other image formats, such as png and gif will work fine as well). If you want your pet to cycle through a series of images, change the number at the end of the image name in the same way we have—so, pet_broken-toe_1.jpg, pet_broken-toe_2.jpg, pet_broken-toe_3.jpg, and so on. Any number of images will work.
