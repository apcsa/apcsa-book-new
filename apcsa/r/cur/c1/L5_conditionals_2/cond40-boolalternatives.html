Here is the code from an earlier step, that assigns a value of "comfortable" or "uncomfortable" to a variable tempStr depending on the value of temp.

if (temp >= 65 && temp <= 75) {          
   tempStr = "comfortable";
} else {
   tempStr = "uncomfortable";
}

Identifying a temperature as "comfortable" or not can be done without && and ||. Here is an example that functions exactly like the above code.

if (temp < 65) {                        
   tempStr = "uncomfortable";
} else if (temp <= 75) {
   tempStr = "comfortable";
} else {
   tempStr = "uncomfortable";
}
