The Java operator for "and" is &&. The corresponding operator for "or" is || (two vertical bars). Here are two program segments that use these operators.

Suppose we want to set the String named tempString to "comfortable" if the value of a variable named temp is between 65 and 75, inclusive, and to set tempString to "uncomfortable" otherwise. Here are a couple of ways to do this. 

if (temp >= 65 && temp <= 75) {
    tempStr = "comfortable";
} else {
    tempStr = "uncomfortable";
}
 
if (temp < 65 || temp > 75) {
    tempStr = "uncomfortable";
} else {
    tempStr = "comfortable";
}
