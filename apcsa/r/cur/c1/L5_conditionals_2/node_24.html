Let's examine another variation of the temperature code:

  • a temperature less than 40 degrees (Fahrenheit) is "cold";
  • a temperature between 40 and 59 is "cool";
  • a temperature between 60 and 79 is "warm";
  • any higher temperature is "hot".

Here is code that implements the specification. 

   if (temp < 40) {
      tempStr = "cold";
   } else if (temp >= 40 && temp < 60) {
      tempStr = "cool";
   } else if (temp >= 60 && temp < 80) {
      tempStr = "warm";
   } else if (temp >= 80) {
      tempStr = "hot";
   }
