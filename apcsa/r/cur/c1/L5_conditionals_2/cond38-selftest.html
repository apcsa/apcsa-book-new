Ages between 13 and 19, inclusive, are teenage years. A program that could accurately identify these would be immediately useful to parents and younger siblings alike.

Open up BlueJ, and open the project for this step.

(The project is lesson5/Age1)

Suppose the variable named age contains an integer between 0 and 100.

In the BlueJ project, fill in the blank so that the program segment, when executed, sets the string variable ageType to "teenage" if age is a teenage year, and to "other" otherwise.
