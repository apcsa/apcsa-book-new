We can use boolean expressions within methods to return true or false. These are often referred to as boolean methods or predicates. We'll look at two examples.

Example 1: Gradebook
A gradebook program might invoke a method every time a user enters a percentage to make sure the entry is valid—that is, between 0 and 100, inclusive. Here's what the method in a GradeBook class might look like: 

// Return true if the given value is a 
//   legal percentage,
// i.e. it's between 0 and 100, inclusive;
//   return false otherwise.
private boolean isLegalPercentage (int val) {
   return (val >= 0 && val <= 100);
}

Here are two ways we could invoke this method to verify that a variable pct contains a legal percentage value: 

   if (isLegalPercentage (pct)) ... 

   boolean isLegal = isLegalPercentage (pct);
