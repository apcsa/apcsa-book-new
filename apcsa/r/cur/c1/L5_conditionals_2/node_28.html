Consider the following method. 

public boolean mystery (int a; int b; int c) {
 if (a == 0) {
 if (b == 0) {
 if (c == 0) {
 return false;
 } else {
 return true;
 }
 } else if (c == 0) {
 return true;
 } else {
 return false;
 }
 } else if (b != 0) {
 return false;
 } else if (c == 0) {
 return true;
 } else {
 return false;
 }
}

In twelve words or less (a symbol like "=" counts as a word), complete the following to produce an informative comment for the method:

The mystery method returns true when ______________________________________, and returns false otherwise.

Making a truth table is probably a useful first step.
