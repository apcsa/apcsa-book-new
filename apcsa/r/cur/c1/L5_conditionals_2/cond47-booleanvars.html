We have seen variables of type int, which can be set to integer values, and variables of type String, which can hold letters.

Similarly, we can have variables of type boolean that can store boolean values, namely true and false. Check it out.

   boolean isFun = true;
   boolean weLike = false;
   boolean whatNow = (temp > 65);
   boolean isComfortable = temp > 65 && temp < 100;

As long as the right-side of the assigment statement evaluates to true or false we can assign it a variable of type boolean.

But even more interesting are boolean methods.

Well-named boolean methods are a tremendous aid for organizing and understanding a program. Typically a name will start with "is" or "has", and is followed by a noun or an adjective. The idea is to have the program read like English.
