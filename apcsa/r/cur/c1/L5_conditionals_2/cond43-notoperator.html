There is one more operator for boolean values, namely the ! or "not" operator. The "not" operator(!) negates the expression that immediately follows it; that is, if the expression evaluates to true, applying the not operator(!) to the expression changes the value to false, and vice versa.

The ! operator is not very useful with numeric comparisons, since other numeric comparison operators are equivalent and more concise (see the table below). We will revisit this operator shortly, however, in the context of object comparisons.

expression that uses ! equivalent expression that doesn't use !
!(x > 5)x <= 5
!(x < 5)x >= 5
!(x >= 5)x < 5
!(x <= 5)x > 5
!(x == 5)x != 5
!(x != 5)x == 5
