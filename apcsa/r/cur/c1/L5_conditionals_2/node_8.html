Consider the outranks method whose header appears below. 

   // Return true if this card outranks the argument, that is,
   // its rank is higher than the argument's rank, or the two 
   // ranks are the same and this card's suit is higher than
   // the argument's suit.
   public boolean outranks (Card c) ...

Two card objects will be involved in the comparison.

  • One of them is the argument c. We can find out the suit and the rank of c by saying c.suit ( ) and c.rank ( ).
  • The other is "this card", the object whose methods we use when we don't specify a dot: suit ( ) and rank ( ). The word this is used in Java to refer to the current object. We can optionally use this with methods or instance variables to make explicit what object we want: this.suit ( ) and this.rank ( ).

Here's how the Card constructor would appear if we use this everywhere it applies. 

   public Card (int rank, int suit) {
      if (this.isLegalRank (rank)) {
         this.myRank = rank;
      } else {
         System.out.println ("Illegal rank: " + rank);
      }
      if (this.isLegalSuit (suit)) {
         this.mySuit = suit;
      } else {
         System.out.println ("Illegal suit: " + suit);
      }
   }
