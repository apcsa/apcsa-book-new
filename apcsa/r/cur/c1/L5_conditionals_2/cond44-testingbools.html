With either an && or an ||, we again should test all possibilities: 

true and/or true
true and/or false
false and/or true
false and/or false

Consider the isLegalPercentage() method from the Gradebook example: 

if (x >= 0 && x <= 100) ...

The cases to test are:

  1. x >= 0 is true, x <= 100 is true
  2. x >= 0 is true, x <= 100 is false
  3. x >= 0 is false, x <= 100 is true
  4. x >= 0 is false, x <= 100 is false

Each case above respresents a set of numbers, defined as follows:
• In the first case, values between 0 and 100, inclusive.
• In the second case, all values greater than 100.
• In the third case, all negative numbers.
• The fourth case contains no values at all and thus doesn't need to be tested.
