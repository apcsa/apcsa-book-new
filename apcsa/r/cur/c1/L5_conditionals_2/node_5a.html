Example 2: CheckerBoard

A checkerboard is 8 rows and 8 columns of squares. One can imagine that a program simulating a checkerboard will need a method to check if a given row and column are legal positions on the board; in others words, within the 8x8 grid. If a user attempts to place a checker at row 10 and column 8 this method should return false and obviously a checker at row 4 and column 2 should return true.

Here's an example of a method in a checkerboard class, that might do just this: 

// Return true if the given coordinates represent
//  a legal checkerboard position
// (i.e. both are between 0 and 7, inclusive); 
//   return false otherwise.
private boolean isLegalPosition (int row, int col) {
   return (row >= 0 && row <= 7
           && col >= 0 && col <= 7);
}
