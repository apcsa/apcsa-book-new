Dot racing is a popular event at American baseball games. In between an inning in the game, the scoreboard shows a computer-generated race between different colored dots. In many venues, people cheer madly for what is, at its heart, a program using a loop and random number generation.

So that you too can feel the adulation of a stadium full of mostly sane fans, complete the DotRacing class below.


Open the current project in BlueJ. (lesson10/DotRacing)

In the program there are three dots, represented by instances of the Dot class, and stored in instance variables racer0, racer1, and racer2. On each turn (a cycle through a while loop, one dot gets to move, and it moves forward a integer number of pixels. The first to reach the finish line wins. You start the race by instantiating an object from the DotRace class.
