This lesson introduced a (pseudo) random number generator, a class named Random in the java.util library. An initialized Random object has a method named nextInt that, given a positive integer argument n, returns an integer between 0 and n-1. Moreover, the sequence of values returned by nextInt looks like each value was randomly chosen.

A random number generator can be used to model real-world actions. For instance, repeated calls to nextInt (2) model repeated flips of a coin, and repeated calls to nextInt (6) simulate the throwing of a six-sided die. Exercises include a random number generator to simulate the "dot racing" game often seen at sports events and the casino game of Craps.
