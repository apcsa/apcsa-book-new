Let's look back at the bar chart you used to show the results from a single die roll, or distributions will six different values. In the GraphGuesser project there is a bar chart class that can show distributions resulting from two six-side dice rolls, with twelve different values. We'll show you some graphs made from (relatively) simple java expressions involving random numbers; your job is to write an expression—we'll call it a generator—that makes a graph pretty similar to the one shown. The graphs were made with 12,000 trials, although you can get the same shape with a different count, of course.

An aside: the BarChart12 class will become much easier to write when you learn about collections, starting in lesson 12. If you look at the source code for the BarChart12 class, you will see thirteen separate instance variables and long, albeit regular conditionals. Imagine writing a bar chart that could show distributions with 100 different values! Collections will make this much, much easier, and even make possible things that are impossible now (for instance, bar charts for any number of values).
