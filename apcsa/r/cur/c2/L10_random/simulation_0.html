When you use randomly generated numbers in your programs, a basic problem arises: how do you test your code? You can't know exactly what will happen as the program executes because, well, it's random!

One way to verify the correctness of the generator is to run it a lot of times, and to judge the results against what you know should be happening on average.

For instance, the following method uses a loop to test coin flips (or, more exactly, generation of random integers between 0 and 1 inclusively). 

   public void tester (int howMany) {
      int k;
      int numHeads;
      k = 0;
      while (k < howMany) {
         k = k + 1;
         if (nextFlip().equals("heads")) {
            numHeads = numHeads + 1;
         }
      }
      System.out.println ("" + numHeads + " out of "
	     + howMany + " flips were heads.");
   }
