The Random class in the java.util library provides a way to produce a sequence of values that appear to be randomly chosen. Random has two methods of interest right now (and many more that can be useful) :

  • a zero-argument constructor, and
  • a nextInt method that takes an argument n and returns an integer between 0 and n-1, inclusive.

Before you can use the Random class to generate a random number you must first instantiate a Random object. For example,

   Random gen = new Random(); 
   int x = gen.nextInt(5);  // returns a random number from 0 to 4

Thus gen.nextInt(2) could represent a coin flip (where a 0 means one side of a coin, and a 1 means the other side). Here's a program segment that uses nextInt: 

private Random coin;
coin = new Random ( );
coinFlip = coin.nextInt(2);

Now consider the expression:

( die.nextInt(6) + 1 )

This represents the throw of a six-sided die (one of the numbers 1, 2, 3, 4, 5, or 6). Here the nextInt method returns a random number from 0 to 5 then 1 is added to this number resulting in a random number from 1 to 6.
