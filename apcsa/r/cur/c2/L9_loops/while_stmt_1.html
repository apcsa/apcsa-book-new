Suppose we want to print the line "I like Java!" eighty times on the console. Doing it once is easy: 

System.out.println ("I like Java!");

Doing it eighty times is woefully slow and inefficient.

A better way to do this in Java is with a while statement. A while statement has three parts:

  1. the keyword while;
  2. a boolean expression enclosed in parentheses; and
  3. a program segment to be repeated, referred to as the body of the while statement.

A while loop is executed as follows.

  1. First, the boolean expression is evaluated. If it's false, the repetition stops, and execution continues with the statement following the while loop body.
  2. If the expression is true, the entire body is executed.
  3. Then the boolean expression is evaluated again, and the while statement terminates or continues as just described.
