On to design of the length method. We start with a counter as in the "I like Java!" printer. It initially gets value 0, indicating that we haven't yet processed any characters in the string.

private int length (String s) {
   int charCount;
   charCount = 0;
   while ( ... ) {
      ...
   }
}

Inside the while loop, we must do two things. One is to increment charCount to acknowledge that we've seen another character in the string. The second is to remove that character from the string to avoid counting it twice or more.

   while ( ... ) {
      charCount = charCount + 1;
      s = s.substring (1); 
      // new s has everything but the first character of old s
   }

The while condition must then ensure that there is at least one more character in the string to tally in the length:

   while (!s.equals("")) {
      charCount = charCount + 1;
      s = s.substring (1); 
      // new s has everything but the first character of old s
   }
