Run-counting

Consider the problem of determining the number of runs in a string. A run is a consecutive sequence of identical characters that's as long as possible. A run may be just one character long. For example, the string "xyyzxxxxmnn" contains 6 runs.

Devise a set of test strings with which to test a run-counting method.

xyyzxxxxmnn (answer 6) xyzxmn (answer 6) xxxx (answer 1) x (answer 1)
typical test: xyyzxxxxmnn boundary case (all runs have 1): xyzxmn boundary case (min # runs): xxxxx boundary case (min # runs, min # chars) consecutive runs with more than 1: xxyy consecutive runs with 1 char each: xy
