Programming environments typically include a debugger. Debuggers vary in the number of features they provide. Almost all, however, include the following:

  • a way to interrupt an executing program at a given statement (this is called setting a breakpoint;
  • ways to examine variable values and execute statements one at a time once the program is interrupted.


In the next few steps, you'll learn how to use a debugger. It is one of the most useful tools you can use when writing a program.
