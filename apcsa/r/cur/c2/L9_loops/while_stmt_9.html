When discussing boolean expressions, we briefly mentioned the exclamation mark "!" ("not") operator. Applied to a boolean expression, it evaluates to true if the expression is false, and vice versa. For example, the following two program segments are equivalent. (Assume that isGood is a boolean method that takes a single integer argument.) 

if (isGood(5)) {
   System.out.println ("5 is good");
} else {
   System.out.println ("5 is bad");
}

if (!isGood(5)) {
   System.out.println ("5 is bad");
} else {
   System.out.println ("5 is good");
}

The best use of ! is not with complicated expressions that use && or ||, but instead with boolean methods whose details have been isolated. For example, the isLeapYear method covered in lesson 4 may be used with ! to determine whether a given year is not a leap year:

expressionvalue
!isLeapYear (1999) true
!isLeapYear (2000) false
!isLeapYear (2012) false
!isLeapYear (2100) true
