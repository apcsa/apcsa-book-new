Open the current project in BlueJ (lesson9/GuesserBuggy).

You run it by calling the constructor, whose argument is the upper limit for the number to be guessed. An argument of 7 should be sufficient.

You set a breakpoint at a particular statement by clicking in the left column by the statement. A stop sign should appear, indicating that execution will stop at the statement when the program is run. A good place to interrupt the program is at the start of the mouseClicked method. Set a breakpoint there, and start the program.
