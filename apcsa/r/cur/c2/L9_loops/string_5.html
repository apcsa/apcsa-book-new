Write a method named zip that, given two strings s1 and s2 of the same length, returns the result of interweaving the characters in the strings.

  • The first character of the result should be the first character in s1,
  • the second character of the result should be the first character in s2,
  • the third character of the result should be the second character in s1,
  • the fourth character of the result should be the second character in s2, and so on...

For instance, a call to

zip("CLFRI", "aiona");

would return the string

"CaLiFoRnIa"


Open the current project in BlueJ (lesson9/Strings3).
