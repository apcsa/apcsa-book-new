In terms of the arguments a and b, what value is returned by the following method? 

private int mystery (int a, int b) {
   int sum;
   sum = 0;
   while (a < b) {
      sum = sum + a;
      a = a + 1;
   }
   return sum;
}
  1. the sum of the integers from a to b, inclusive
  2. the sum of the integers from a to b-1, inclusive
  3. the sum of the integers from a+1 to b, inclusive
  4. the sum of the integers from a+1 to b-1, inclusive
