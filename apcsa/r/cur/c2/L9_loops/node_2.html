Here's the original code: 

count = 0;
while (count < 80) {
    count = count + 1;
    System.out.println ("I like Java!");
}

Suppose the statements in the body are switched as shown below. 

count = 0;
while (count < 80) {
    System.out.println ("I like Java!");
    count = count + 1;
}

Does the output remain the same? Explain why or why not.

The second version prints an extra "I love Java!" line, since count isn't increased until after the line is printed rather than vice versa.
The two versions print the same thing.
