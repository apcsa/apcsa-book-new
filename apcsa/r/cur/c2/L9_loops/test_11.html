Test the following implementation of the number guesser, which has the computer doing the guessing.

  • The constructor for the Guesser class has an int parameter that defines the range of integers to guess. It then draws three "buttons" on which to specify that the guess made by the computer is too big, too small, or the number to be guessed.

  • The computer's guess is displayed in the first call to printComponent. From then on, the computer will determine its next guess based on response by the user. The mouseClicked method indicates whether the guess is too small, too large, or a correct guess, and produces the next guess based on the position of the click.

Open the current project in BlueJ (lesson9\NumberGuesser).
