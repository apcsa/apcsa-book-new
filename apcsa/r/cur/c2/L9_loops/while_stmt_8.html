Observe what happens if you omit the increment of a. 

private int mystery (int a, int b) {
   int sum;
   sum = 0;
   while (a < b) {
      sum = sum + a;
      a = a + 1;
   }
   return sum;
}

The modified program continues forever, since without the increment, nothing brings a closer to b. This is called an infinite loop.

When you code a loop, make sure that some statement in the body always makes progress in getting closer to where the loop should terminate.


Think about it for a second. If we don't increment a, then we are just doing the same comparison over and over again.
