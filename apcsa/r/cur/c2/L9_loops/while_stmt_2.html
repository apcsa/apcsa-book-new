To print the line "I like Java!" eighty times on the console, we use a variable named count (any reasonably descriptive name is fine) whose value says how many lines we've printed so far. Thus count plays the role of a counter variable or a "stepper" variable. The initial value of the int count should be 0, indicating that we haven't printed anything yet. 

   count = 0;

The loop body contains the println statement as well as an assignment statement that increments count. 

   count = count + 1;         
   System.out.println ("I like Java!");

Finally, we supply the boolean expression. We want to continue the loop while we haven't yet printed eighty lines. Alternatively, we want to terminate the loop once we've printed the eightieth line. A boolean expression that represents our desires is 

   while (count < 80)

Putting it all together, we have 

   count = 0;
   while (count < 80) {
      count = count + 1;
      System.out.println ("I like Java!");
   }

Quite an exchange—we need only five lines to do the work of eighty!
