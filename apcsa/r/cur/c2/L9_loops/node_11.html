In terms of the arguments a and b, what value is returned by the following method? 

private int mystery (int a, int b) {
    int sum;
    sum = 0;
    while (a < b) {
        sum = sum + a;
        a = a + 1;
    }
    return sum;
}
the sum of the integers from a to b, inclusive
No, the while loop will stop when a equals b, and the loop body won't be run.
the sum of the integers from a to b-1, inclusive
Right!
the sum of the integers from a+1 to b, inclusive
Nope. The first time sum is increased by a, not a+1.
the sum of the integers from a+1 to b-1, inclusive
No. The first time sum is increased by a, not a+1.
