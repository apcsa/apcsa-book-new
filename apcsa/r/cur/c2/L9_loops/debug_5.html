Consider the following class. 

public class TestString {
    private String myS;

    public TestString(String s)
    {
        myS = s;
    }
    
    // Return true when myS is the result of inserting exactly one character
    // into the argument; return false otherwise. For example, suppose that
    // myS is "abcd"; contains1moreThan should return true for arguments "acd"
    // and "bcd" and false for arguments "ab", "xcd", and "abcd".
    public boolean contains1moreThan (String s) {
        if (myS.length() != s.length()+1) {
            return false;
        } else {
            int k=0;
            while (k < myS.length()) {
                if (!myS.substring(k,k+1).equals(s.substring(k,k+1))) {
                    break;
                }
                k++;
            }
            return myS.substring (k+2).equals(s.substring(k+1));
        }
    }
}

The contains1moreThan method has a bug. Find and fix it.
