The String class that we used to store parts of a name has its own length method. It is illustrative, however, to program it here using a while loop. We'll assume that our length method is inside the Name class, and will be used with one of the components of the name in the same way as the fixedCase method.

First, recall some facts about the substring method.



  • s.substring (k) returns a new string that contains the characters starting at position k of s and extending to the end of s. For example, if s contains "I love Java!", s.substring (2) returns "love Java!".
  • s.substring (startPos, endPos) returns a new string that contains the characters starting at position startPos of s and extending up to but not including the character at position endPos. For example, if s contains "I love Java!", s.substring (2, 6) returns "love". Another example: s.substring (k, k+1) returns a single-character string that contains the character at position k of s.
