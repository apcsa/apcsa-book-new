The % can be used to examine individual digits of a number. For example, 1234567 % 10 is 7, and 1234567 / 10 is 123456.

Some validation schemes for credit card numbers, license numbers, etc. involve a check digit that's part of the number. If the check digit is what results from doing some arithmetic on the remaining digits, the number is legal, otherwise not. (See http://en.wikipedia.org/wiki/Check_digit for more information.) This exercise involves a method that might be used to help validate a number, namely adding the digits of an integer.

Write a method named digitSum that adds the digits of its positive integer argument and returns the sum.
