Java comes with thousands and thousands of pre-written classes. Learning to use these classes is a big part of learning to program in Java.

The two libraries imported in the greeting program are: 

import java.awt.*;
import javax.swing.*;

Among the classes supplied in those two libraries are the JFrame and JPanel classes. When doing Java graphics, these are two very common and useful classes.


The relation of JFrame to JPanel is similar to that of an easel to a canvas. All the drawing is done on the JPanel object (the canvas), and the JFrame (the easel) acts as the window that holds the JPanel.
