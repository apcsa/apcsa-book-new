(The project is lesson7/Greetings2)

The drawLine method takes four arguments:

  1. the column coordinate of the start of the line;
  2. the row coordinate of the start of the line;
  3. the column coordinate of the end of the line; and
  4. the row coordinate of the end of the line.

Note the difference between the last two parameters in drawLine and drawRect.

Write a newDrawRect method that consists of four calls to drawLine. Try it out and see if it works.
