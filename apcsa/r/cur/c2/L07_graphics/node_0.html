So far, you've learned a lot about how to write Java programs that do smart things with numbers and strings. But, so far, the only way to tell a person about what is going on is by printing a number or string to the Java console. This next collection of activities will deal with the graphics facilities of Java. We'll be creating shapes, artwork, and diagrams on the screen, and learning some interesting programming techniques at the same time.
