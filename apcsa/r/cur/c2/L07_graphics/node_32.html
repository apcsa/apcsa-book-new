Turtle graphics is an easy way of graphics programming, popularized in the programming language "Logo". There are many different kinds of turtle graphics, but there are some basic features always present. A programmer gives commands to a 'turtle' that then walks around the drawing canvas. A command such as walk might move the turtle forward. The turtle carries a pen that can be down or up, which, when down, will leave a trace of where the turtle has walked. Using turtle graphics, some relatively complicated pictures can be drawn with relatively simple programs.

In this exercise, you will create a Turtle class that can be used to do some simple turtle graphics.

Your Turtle class should have four instance variables:

  • the first representing the direction in which the turtle is facing ("up", "right", "left", or "down")
  • the second representing the state of the pen (up or down)
  • and the final two keeping track of the coordinates of the pixel that the turtle is currently on.

Choose some reasonable starting values for these variables.

It should have five methods, all taking no arguments:

  • lowerPen and raisePen, which lower and raise the pen, respectively;
  • takeStep, which moves the turtle 20 pixels in the direction it is headed, drawing a line or not according to whether the pen is down or up.
  • turnRight, which changes the direction of the turtle according to the following table:
    before the call to turnRight after the call
    up right
    right down
    down left
    left up
  • A turnLeft method, logically opposite to the turnRight method above.

Test your program by making another class called Drawing1. In this class, make a Turtle object and give it some commands.
