A user is allowed to resize a window while a program is running, so our programs need to be flexible in that way. Java gives us some help here with the getSize method. getSize returns a Dimension object that has two public instance variables, width and height, representing the current size of this JPanelobject.

For instance, a call of getSize().width inside a method in a JPanel class will return the width of the JPanel as an int.

The expression getSize().width is relatively complex. Here is how it's evaluated:

  • getSize is a method of the JPanel and therefore of our Greetings class. We call it.
  • It returns an object of class Dimension.
  • A Dimension object has a width instance variable, which we access via the "dot" operator. That gives us an int.
