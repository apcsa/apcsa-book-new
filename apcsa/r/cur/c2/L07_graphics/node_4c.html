public void paintComponent (Graphics g) {
      g.drawString ("Hi there!", 150, 200);	
   }

What is the Graphics g parameter of the paintComponent method?

It's just an object from another class (the class Graphics) defined inside one of the Java libraries. It needs to be used with the paintComponent method. When Java decides that paintComponent should be called, it will make the Graphics object for you.



What is the Graphics object used for?

The Graphics object is what knows how to do the actual drawing (that is, has the drawString method, among others).
