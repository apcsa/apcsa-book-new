Java comes with lots of class libraries. You can think of these like normal libraries, but instead of containing books they contain already written Java classes.

To write a program that draws on the screen, we will use the classes from two of these libraries: java.awt (short for Abstract Windowing Toolkit) and javax.swing (an extension of java.awt).

The import keyword is used to access particular libraries, and needs to be at the top of the program. Our greeting program used the statement

import javax.swing.*;

which gives you access to all the classes in the javax.swing library. The * (an asterisk) tells the compiler to provide access to every class in the library. You could, instead of using an asterisk, name the classes individually with multiple import statements.
