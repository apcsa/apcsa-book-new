The drawRect method of the Graphics class draws a rectangle on a JPanel. It takes four arguments:

  1. the column coordinate of the rectangle's upper left corner;
  2. the row coordinate of the rectangle's upper left corner;
  3. the width in pixels of the rectangle; and
  4. the height in pixels of the rectangle.

The picture to the right displays the result of adding the call to the Greetings class:

public void paintComponent (Graphics g) {
   g.drawString ("Hi there!", 150, 200);	
   // draw a rectangle with its 
   // upper left at (150,200), 70 pixels
   // wide, and 25 pixels high
   g.drawRect (150, 200, 70, 25);     //ADDED
}
