The drawRect method draws an outline of a rectangle. The fillRect method takes the same arguments as drawRect, but produces a filled (solid) rectangle.

To draw a circle or ellipse, use the drawOval method. For a filled circle or ellipse, use fillOval. Both take the same arguments as drawRect. The first two arguments specify the upper left corner of the bounding rectangle of the ellipse, and the next two arguments specify the bounding rectangle's width and height.

The default color in which lines and fills are drawn is black. This can be changed, however, using the Graphics method named setColor. The argument is one of the following: 

Color.black
Color.blue
Color.cyan
Color.darkGray
Color.gray

Color.green
Color.lightGray
Color.magenta
Color.orange

Color.pink
Color.red
Color.white
Color.yellow

For example, the following code 

   public void paintComponent(Graphics g) {
      g.setColor (Color.pink);
      g.drawString ("Hi there!", 150, 200); 
   }

will print our greeting in pink! A call to the getColor method, which takes no arguments, returns the current drawing color. Click on the Graphics link to the right for a summary of commonly used Graphics methods.
