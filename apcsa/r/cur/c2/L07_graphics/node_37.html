One way to use getSize is to change our Greetings program to put the "Hi there!" message in the middle of the window, however big it is. Here's the code for the modified paintComponent:

public void paintComponent (Graphics g) {
   g.drawString ("Hi there!", getSize().width/2, getSize().height/2);
}

Copy the code above into the Greetings class paintComponent method. Run the program and test that it works by resizing the window by hand.
