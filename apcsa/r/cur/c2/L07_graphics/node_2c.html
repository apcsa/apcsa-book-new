The import statement gives us access to classes in a library. The extends keyword is one way to use those classes.

By saying that one class extends another, we mean that an object of the second class will act just like an object of the first class except for a few "extensions". Generally, these extensions are additional or redefined methods. Consider the beginning of the Greetings class from earlier:

public class Greetings extends JPanel {

This code is saying that objects from our Greetings class will act just like JPanel objects—in particular, we can draw on them—with some minor differences.

This technique is called inheritance, and you don't need to understand it fully now. We'll get a lot more practice with it later in the course.
