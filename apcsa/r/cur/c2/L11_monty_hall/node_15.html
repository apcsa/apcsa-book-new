This activity involves two simulations of the Monty game, one in which the contestant always sticks with her current choice, the other in which she always switches. Here's how to do this.

  1. First, change the guess2 method to always stick with the current choice.
  2. Next, supply a version of the Controller.java program that replaces the bodies of the constructor and the say, picture, open, and close methods by empty bodies and the doorSelected method by return 0.
  3. Then provide a class named SimulateMonty that contains a method simulate. The simulate method takes one argument, the number of games to simulate. For each game, it will call the Monty constructor, then repeatedly call playGame, keeping track of the number of wins.
  4. After playing all the games, it should print the percentage of wins with the "always stick" strategy.

Now change the guess2 method to always switch, and rerun the simulation.
