Simulate the clicking of a two-digit counter. The drawing window should appear as follows:

7 9 ^ V

Clicking on the "^" should increase the displayed value by 1. For instance, if the current counter was as shown above, clicking on the "^" would change the counter to 8 0.

Conversely, clicking on "V" should decrease the displayed value by 1. The displayed value should "wrap around": incrementing 99 should produce 00, and decrementing 00 should produce 99.

A tip: keep track of the current counter value in an int instance variable. You can determine the left digit by dividing by 10, and the right digit by using the remainder, or modulus, operator: %.
