Wait, I have to use extends at some points and implements at others?

What is the difference between the two? Is there any rhyme or reason to Java!?

Well, yes, if there is one thing Java has, it is a reason for everything you must do.

Both extends and implements do similar things in your program, helping you use other classes you might not have written.

For now, just remember that you extend a JPanel, but implement a MouseListener.

There are differences between what these two keywords do. You'll learn all about them and how they work in a few weeks when we look at inheritance.
