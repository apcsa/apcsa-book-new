You said there were two things I need to do to handle mouse events. What else is there?

Right. You use implements to tell Java that your class can deal with mouse events. But, you also need to tell Java that you want those events sent to your mouse methods.

You do this by calling the method addMouseListener, usually in the constructor for your class: 

public SimpleClicker() extends JPanel implements MouseListener ... {
      ...
   addMouseListener (this);
      ...
}

This is called registering your object for mouse events. (Remember, in the constructor, this refers to the new object being created.)
