You have probably noticed that each of the mouse methods (e.g., mouseClicked) gets a single argument of the class MouseEvent. This object contains information about the mouse click that might be useful.

The MouseEvent object has two methods that allow us to determine where in the drawing window the mouse was clicked, pressed, or released: getX and getY. The getX method returns the column coordinate and the getY method returns the row coordinate.

Copy the Clicker class into a new class called CircleClicker, and modify it so that for each click it draws a red circle at the location of the click.
