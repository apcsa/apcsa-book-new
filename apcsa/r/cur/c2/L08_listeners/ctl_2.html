Let's create some buttons to help control a turtle graphics widow. The turtle graphics engine was presented as an exercise in Lesson 7, near the end.

You'll need to draw a control panel at the top of the window:

forward turn right raise pen

When first initialized, the graphics window should be blank. When one of the buttons in the control panel is clicked, the relevant command should be given to the turtle. For instance, if "forward" is clicked, then the turtle should move forward in its current direction, drawing a line if the pen is down.

The text of the 'last' control should change based on the state of the pen. If the pen is down, it should appear as above, containing "raise pen". When the pen is up, "lower pen" should appear.

The turtle should not be allowed to move off the screen or into the control panel.
