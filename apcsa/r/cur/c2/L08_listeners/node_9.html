Much of the program resembles the graphics code we saw in lesson 7. For example, we see 

public class class name extends JPanel ...

as before. The first few statements of the constructor should be familiar as well. To access the graphics classes, we used import statements at the start of the program. We add another import statement to access mouse event handler methods. The paintComponent is what draws on the screen as in earlier code.

A feature of Java that we saw for the first time here is an interface. A Java interface sets up a contract in which the user of the interface promises to set up a given set of callback methods. We use the implements keyword to agree to that contract. Here, we agree to a contract about handling mouse clicks, promising that we will supply methods mouseClicked, mouseReleased, mousePressed, mouseEntered, and mouseExited. (We only actually use mouseClicked.)

Finally, here are two more details. We connect the listener methods to the Java runtime system with the statement 

addMouseListener (this);

placed in the constructor. When the mouseClicked method is called, it can call repaint to redraw the screen.
