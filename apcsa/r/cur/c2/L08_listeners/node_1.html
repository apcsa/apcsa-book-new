The mouse event handling that we are going to start with involves several callback methods:

  • mousePressed handles a press of the mouse button;
  • mouseReleased handles a release of the mouse button;
  • mouseClicked handles a mouse click (a press and release at the same spot);
  • mouseEntered handles a move of the mouse to this drawing window;
  • mouseExited handles a move of the mouse from inside this window to outside this window.

(We will concern ourselves mainly with the first three of these.)

All of these methods are callbacks. Like the paintComponent method in earlier activities, we never call these mouse event handlers ourselves. Instead, the underlying Java system calls the relevant event handler when the event happens.

So, for instance, you write the method mouseClicked, and it will be called every time the mouse button is clicked.
