Let's look at a working example of the SimpleClicker class.

Open the current project in BlueJ.

(The project is lesson8/SimpleClicker.)

The project consists of a single class SimpleClicker. Look at the source code for that class, and notice:

  • The class both extends JPanel and implements MouseListener.
  • There is another import statement necessary for using MouseListener: 
    import java.awt.event.*
  • The last line of the constructor adds the mouse listener to the SimpleClicker object being created.
  • It fulfills the contract of the MouseListener interface, because it has the five required methods.
  • Only one of those methods, mouseClicked, actually has a method body. The other methods have to be there to fulfill the interface contract, but they don't do anything.
