Some video games keep track of the highest scores that players have achieved. This exercise involves implementing such a feature.

We start with a simple class that represents a person and the score he or she has received. 

public class Score {
   
   private int myPoints;
   private Name myPlayerName;

   public Score (int points, Name playerName) {
      myPoints = points;
      myPlayerName = playerName;
   }

   // Return true if this Score's myPoints is greater than
   // the argument Score's myPoints; return false otherwise.
   public boolean isBetterThan (Score score) {
      ...
   }

   public String toString ( ) {
      return myPoints + "\t" + myPlayerName;
   }
}

Now define and test a TopTen class whose single instance variable is an ArrayList of Score objects. This class provides the following methods:

  • A constructor with no arguments.
  • A void method named update that takes a Score and adds it to the score collection.
  • A void method named print that prints the top ten Score objects, ordered from highest to lowest points. If there are fewer than ten scores in the collection, print should print as many as are currently in the collection.
