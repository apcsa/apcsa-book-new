The for statement provides another way in Java to repeat a given program segment. It starts with the word "for", continues with loop information inside parentheses, and ends with the segment to be repeated—the loop body—enclosed in curly braces. 

for (initializations ; test; increments){
   ...
}

Loop execution proceeds as follows.

  1. First, the initializations are performed.
  2. Then the test is evaluated. If it's false, the loop is finished and execution continues with the following statement. If the test is true, the loop body is executed.
  3. Then the increments are performed.
  4. The test is evaluated again; a failed test halts the loop, while a successful test results in another execution of the loop body.

The following would print the first name of every Name object in the friends list, assuming the Name object has a method firstName() defined to return the first name. 

// Print the initials of the names in friends.
for (int x = 0; x < friends.size(); x++) {
   System.out.println (friends.get(x).firstName());
}
