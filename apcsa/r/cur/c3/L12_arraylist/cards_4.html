It is also useful in Poker to determine the most often-occurring rank in the hand:

  1. Does the hand contain four cards, all of the same rank? If so, return 4.
  2. If not, does the hand contain three cards, all of the same rank? If so, return 3.
  3. If not, does the hand contain two cards of the same rank? If so, return 2.
  4. Otherwise return 1.

One approach to computing this value is to check how many times each rank occurs in the hand. Open the BlueJ project for this node and take a look at the pseudocode in the maxInRank in the Hand class.
