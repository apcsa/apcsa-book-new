We also worked with another looping construct, the for statement. There are two kinds of for statement: 

for ( initialization,
      continuation test,
      update ) {
   statements to repeat
}

for ( iteration variable : ArrayList ) {
   statements to repeat
}

The statements to repeat comprise the loop body. In the first form of for loop, we specify initializations necessary to start the loop, a boolean expression that says whether to execute the body one more time, and statements that update loop variables. (These often involve incrementing using the ++ operator or decrementing using the -- operator.) The second form assumes that a collection is involved, and executes the body for each element of the collection.

We also encountered several patterns for collection processing:

  • processing every element of the collection;
  • searching for a collection element that satisfies a given condition; and
  • processing every pair of consecutive sequence elements.

Instances of these patterns will appear over and over in subsequent lessons.
