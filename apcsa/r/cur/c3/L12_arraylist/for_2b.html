There is a variation on the for loop, named a for-each loop. It's used with collection class objects, and basically is a translation of "for each element of the collection, ..." In pseudocode, it looks like the following: 

for (ElementType variable : collection) {
   ...
}

Two examples of its use with the friends and clicks collections appear below. 

// Print the initials of the names in friends.
ArrayList friends = new ArrayList();
//... add a bunch of Name objects to friends

for (Name person : friends) {
   System.out.println (person.initials());
}

// Draw a circle at each click position.
ArrayList clicks = new ArrayList();
//... add a bunch of Point objects to clicks

for (Point p: clicks) {
   g.fillOval (p.x-5, p.y-5, 10, 10);
}
