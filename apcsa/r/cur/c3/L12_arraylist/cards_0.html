A computer implementation of the card game of Poker includes numerous examples of array processing. A Poker program would include a Card class whose methods include the following:

  • int rank ( ), which returns an int between 2 and 14, inclusive (11 represents a jack, 12 a queen, 13 a king, and 14 an ace);
  • int suit ( ), which returns an integer from 1 to 4 (representing the four suits—clubs, diamonds, hearts, and spades);
  • a zero-argument constructor, which initializes a Card with randomly chosen rank and suit.

A poker hand or a deck of cards would then be an ArrayList<Card> object.
