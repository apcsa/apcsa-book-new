The ArrayList in which clicks are collected is named myCorners. It's declared with the myWindowWidth and myWindowHeight variables, and initialized in the Polygon constructor to be an empty ArrayList.

Each successive pair of mouse click coordinates is stored in a Point object (Point is declared in java.awt; it has public instance variables named x and y.) After saving the most recent click, the program draws the partially completed polygon. If the polygon is complete—the current click is sufficiently near any of the earlier clicks—the program empties myCorners and continues.
