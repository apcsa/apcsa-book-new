Code you wrote for the paintComponent method most likely contains an example of a common ArrayList pattern, that of processing every element in the ArrayList. A variable is initialized to 0, then incremented each time through the loop in order to process element 0, element 1, and so on up to element size()-1. In pseudocode, it looks like the following: 

k = 0;
while (k < the size of the ArrayList) {
   process element k of the ArrayList;
   k = k + 1;
}

Code you wrote for the mouseClicked method probably involved search in an ArrayList, another common pattern. The code resembles that of processing every element—where "processing" means "checking whether the element is what we're searching for"— except that it is possible for the loop to end early if the search succeeds. Exiting the loop is done with the break statement. Here is pseudocode. 

k = 0;
while (k < the size of the ArrayList) {
   if element k of the ArrayList is what we're searching for) {
      break;
   }
   k = k + 1;
}
