The ArrayList class has several methods. For now, we will look at three:

  • get, which returns the element at the given position in the ArrayList;
  • size, which returns the number of elements contained in the ArrayList; and
  • add, which adds its argument to the end of the ArrayList.

We start by instantiating an ArrayList of Name objects: 

   friends = new ArrayList<Name> ( );

friends.size() returns 0.   friends.get(0) is trying to access the 0th element of friends, but causes an exception—"array index out of bounds"—because friends is empty and has no element at position 0.
