We continue by adding two more names to the ArrayList: 

   friends.add (new Name ("Hannah", "Montana"));
   friends.add (new Name ("Mickey", "Mouse"));
   Name n1 = friends.get(1);
   Name n2 = friends.get(2);

After these statements are executed, we have an ArrayList whose size is 3, along with two references n1 and n2 to Name objects. Also notice that the get method does not change the ArrayList.
