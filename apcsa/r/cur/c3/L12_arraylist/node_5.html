The add method puts its argument at the end of the ArrayList and increases the size by 1. Here's an example, using the ArrayList of names just introduced: 

   friends.add (new Name ("SpongeBob", "SquarePants"));

friends.get(0) would return the name we just added; get with any other integer argument would cause an "array index out of bounds" exception.

The argument to add should be an initialized object. In the example just given, that object is created on the fly, since new produces an initialized object. The example could also have been written as follows: 

   Name myPal = new Name ("SpongeBob", "SquarePants");
   friends.add (myPal);
