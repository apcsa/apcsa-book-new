Open the Polygon1 project (lesson12\Polygon1).

This program intends to collect positions of mouse clicks and draw lines that connect each consecutive pair of clicks in order to form a polygon. More specifically, you should add code to do the following:

  • declare an ArrayList of Point objects;

  • supply a body of the mouseClicked method in which the position of the click is stored in a Point object and added to the ArrayList, and ended with a call to repaint;

  • complete the paintComponent method with code that draws a filled circle at each click position and connects consecutive clicks with a line as shown below.

A good way to draw the mouse clicks is 

Point p = ...;
g.fillOval (p.x-5, p.y-5, 10, 10);
