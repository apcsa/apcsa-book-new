Remember the DotRacing program? Use ArrayLists to allow any number of racers.

Open the DotRacingArrayList project (lesson12/DotRacingArrayList).

The Dot class hasn't been changed from the project you used way back in lesson 10, but the DotRacingArrayList class is different in several ways from the DotRacing class you used then. The main difference is the dots instance variable, which is an ArrayList that holds Dot objects.

You'll need to finish three methods to make this program work correctly:

  • The initializeRacers method needs set up dots correctly. The main loop has already be done for you.
  • The whoMoves method needs to return a randomly choosen Dot object.
  • And, the drawRacers method needs to call drawRacer for each Dot object in dots.

After you have finished the program, test it by changing the number of racers in a race—this is specified in the initializeRacers method.

You might be interested in the randomName and randomColor methods, which set up ArrayLists and randomly pick an element from each. Can you modify them so that duplicate names and/or colors aren't choosen in a race? This might be tricky!
