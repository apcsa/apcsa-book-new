The current Adventure program has several rooms in the world. These are created in the World class with the base class Room. For instance, the code 

Room corridor = new Room(
  "The corridor", 
  "You're in a corridor connecting to the kitchen and the bedroom. The carpets are \n" +
  "stained, the light fixture in the ceiling is a bare bulb, and there are several \n" +
  "locked doors along its length.");
addRoom(corridor);
corridor.addExit(bedroom, "north");
corridor.addExit(kitchen, "south");

adds the room Corridor to the game. Note that the description is very long, and to make it fit into the editor window it is constructed from several strings. The \n at the end of each string forces System.out.println to start a new line (think "n for newline").

Objects made from the Room class will all have the same functionality, and only differ in their name and descriptions. Lets see what we can do by extending the Room class.
