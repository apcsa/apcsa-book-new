The Command class processes the users input, after methods defined in its parent class InputGather prints the prompt and puts each word of the input the ArrayList inputWords. If the user inputs go north, which of the following Command methods will return true?
isInventory()
No, this method will only return true for input such as inventory or inv. The method isInventory checks whether the first word of the input is in the array of strings INVENTORY_words
isHelp()
No, this method will only return true for input such as help me. The method isHelp checks whether the first word of the input is in the array of strings HELP_words
isTravel()
Correct. Because the first word of the input (go) is in the array TRAVEL_words, isTravel() will return true.
firstWordIn()
This is a utility method, and it needs to receive a array of Strings to whether the first word of the input is in that array.
getDirection
getDirection doesn't return a boolean, but a string. With the given input, getDirection will return north
