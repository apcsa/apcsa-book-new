Let's check out some sounds in Java.

Open the current project in BlueJ (lesson13/Sound).

The project in BlueJ contains several classes, but you only need to pay attention to the Sound class. This is where you will add your methods to transform the sound (methods like addReverb andincreaseVolume, for example).

Before looking at the code for the Sound class, let's look at a sound:

Right-click on the Sound class to create a new instance. You will see three different constructors. For now, choose the one with the String filename argument—use this one any time you know the name of the file you want to open and can type it as a string.

Type in, exactly, "sounds/birdcall.wav" when BlueJ asks you for the argument. Note: you need to include those double quotation marks at the beginning and the end. You are making a Java String object.

You new window will pop up containing a very jagged image, something like the image to the right. This is the sound viewer, and shows the pressure waves for the sound. You can think of it as a viewer of ArrayLists—very large ArrayLists containing integers—because that is the same thing as pressure waves, as far as we are concerned!

Click the play button. You should hear a bird call. If you don't, make sure the volume setting in your computer is loud enough.
