People have historically be superstitious about the ability to speak backwards. But, this is before they had digital recordings of sound and Java programs that manipulate those sounds with the greatest of ease.

Write reverse. Simply put, you need to swap the first and last individual samples, the second and second to last, and so forth.

Test your code by listening to several different sounds reversed. (We have provided many different files inside the sounds directory inside your BlueJ project. Try using the Sound constructor that takes no argument: it will open a dialog box for you choose from.

Another test: a sound that is reversed twice should be exactly the same as the original sound.
