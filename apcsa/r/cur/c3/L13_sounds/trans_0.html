When we want to change a sound file in some way, we need to write a method in the Sound class that modifies the numbers in the ArrayList stored in the myData instance variable.

The easiest transformations are those that modify each invidual sample in the same way.

Write a method amplify that multiplies every individual sample in a sound by 1.2. This will make each sample slightly bigger if it is positive, and slightly smaller if it is negative. This should make the sound louder.

  • You will be operating on the ArrayList reference field myData, using the get method to read the original value and the set method to write a new value to that same place.

    You haven't seen an ArrayList's set method yet: it takes an index and a value, and sets the cell at that index to the value you give it.

    For example, to increase the 10th element in the list by 1.2. You would write:

                   myData.set(9, (int)(myData.get(9) * 1.2));

    You might be wondering what (int) does? This is called casting; it forces the value behind it to become an int -- we'll look at this more in the next step.

  • You'll need to write a loop that runs the same code on each value in the ArrayList.
  • When you are done, you will need to call refresh on the Sound object to see your changes.

Compile the project, load the birdcall.wav file, and listen to it.

Run your amplify method on it. Listen again: did it get louder? Run amplify again and perhaps again, listening each time.
