To declare a static constant, we use the keyword static. A constant may be either public or private, depending on whether or not it should be accessible from outside the class.

To declare a constant, we use the keyword final and use an initializer—an equals sign, followed by an associated value—to provide the constant's value. Assignments are not otherwise allowed to an identifier declared as final.

By convention, names of constants are written in all capital letters.

Thus, here's how a constant representing the number of suits would appear in a Card class: 

public static final int NUMSUITS = 4;

Within the class in which it's declared, a static constant may be accessed by merely using its name, in the same way as instance variables. From outside the class, we attach the class name and a dot to the front of the name, for example, 

g.setColor (Color.RED);
