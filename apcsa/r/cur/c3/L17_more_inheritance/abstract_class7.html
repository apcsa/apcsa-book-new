Modify the Measurement class so that it can be used to sort the elements of an ArrayList. Do this by

  • implementing the interface Comparable <Measurement>, and
  • supplying a body for the compareTo (Measurement) method.

The compareTo method returns a negative value if this Measurement is less than the argument, a positive value if this Measurement is greater than the argument, and 0 if they're equal.

The class below can be used to test your revisions. 

import java.util.*;

public class MeasurementTest2 {

   public MeasurementTest2 ( ) {
      ArrayList<Measurement> a = new ArrayList<Measurement> ( );
      a.add (new Measurement (3, 5));
      a.add (new Measurement (10, 9));
      a.add (new Measurement (6, 3));
      Measurement m1 = new Measurement (2, 7);
      a.add (m1);
      Collections.sort (a);
      System.out.println (a);
   }
}
