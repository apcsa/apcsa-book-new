In Java, such a placeholder method is called an abstract method. It is specified with only a header: 

   public abstract boolean cooperates ( );

A class that contains an abstract method must be declared as an abstract class, using the abstract keyword. 

public abstract class Player {

   private int myScore;

   public Player ( ) {
      myScore = 0;
   }

   public void increaseScore (int amt) {
      myScore += amt;
   }

   public int score ( ) {
      return myScore;
   }

   public abstract boolean cooperates ( );

   public void remember (boolean opponentChoice) {
   }
}

Note that the methods dealing with the score are common to all players and are coded normally.

We would then have the always-cooperating player using inheritance as we did with the other player classes: 

class Cooperator extends Player {

   public boolean cooperates ( ) {
      return true;
   }
}
