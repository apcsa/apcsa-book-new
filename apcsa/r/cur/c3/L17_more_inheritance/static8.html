BlueJ makes it easy to start a program by instantiating an object. In order to run a program outside BlueJ, however, we must supply a static method that does whatever object instantiation is appropriate. This method is named "main"; its signature is 

public static void main (String [ ] args)

(We'll explain the argument to main in a couple of weeks.)

A Java program is run outside BlueJ by giving the name of a class. The Java system looks for a suitably defined main method in that class. If main exists, it is run; otherwise an error is signalled.

The main method must be static because it is run before any objects are instantiated.
