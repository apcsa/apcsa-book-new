Static variables actually most commonly appear as static constants, quantities that are assigned once and never changed thereafter. For example, the various colors in the Color class (BLACK, WHITE, RED, etc.) are static constants.

As noted before, static quantities are associated with a class rather than with individual objects. A class representing bouncing objects might have a static constant representing the value of gravity, since this constant would apply to all the bouncing objects. A class representing standard playing cards might have a static constant saying how many suits there are.
