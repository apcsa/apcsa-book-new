Suppose we want to have a class that's just like Refillable except that the maximum amount it can hold is limited. We'll call this class LimitedRefillable.

What extra code must LimitedRefillable provide?

There may be more than one correct answer

an instance variable to keep track of the limit
Correct. Since the new class must store the limit, it will need its own instance variable to do so.
a constructor
Correct. The new constructor will call the Refillable constructor, then initialize the limit variable.
useUp
No, Refillable's useUp method will work fine for LimitedRefillable.
refill
Correct. The refill method in LimitedRefillable will check that the added amount doesn't exceed the limit, and if so it will call Refillable's refill method.
currentAmount
No, Refillable's currentAmount method will work fine for LimitedRefillable.
