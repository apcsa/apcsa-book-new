Consider a Counter class that provides several public methods.

  • a constructor that initializes the counter to 0;
  • a void increment method that adds 1 to the counter's value;
  • a void reset method that resets the counter to 0; and
  • an int value method that returns the counter's value.

Now suppose we want to define a "mod N" counter, whose value cycles from 0 to N-1 and back. There are lots of real-life examples of mod N counters including clocks, automobile odometers, and television channel selectors. Provide a definition of a ModNCounter class that extends the Counter class. Take advantage of the Counter methods as much as possible rather than reinventing the wheel; in particular, your ModNCounter class should only have one instance variable.
