Inheritance relationships throughout Java form a hierarchy, with the most general class—named Object—at the top. Every class definition that doesn't explicitly extend some other class implicitly extends the Object class.

For example, 

public class Refillable { ... }

Since Refillable does not extend anything explicitly it automatically extends Object.

Likewise, the java.util class library provides a hierarchy of classes called the Collection hierarchy that is rooted at the Object class and includes the ArrayList class.
