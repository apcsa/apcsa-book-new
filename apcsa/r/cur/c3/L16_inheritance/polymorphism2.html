An especially convenient application of polymorphism is the toString method. toString is called by System.out.print and System.out.println, and thus is useful for debugging and testing a class. It's a method of the Object class, so every class has a toString method available for System.out.print and System.out.println to use.

Unfortunately, the toString method provided by the Object class isn't very useful. It returns a String that says where the object resides in memory. That makes it a great candidate for overriding.
