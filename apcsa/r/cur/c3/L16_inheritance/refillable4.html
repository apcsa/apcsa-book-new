The inheriting class is not allowed to access any private data or private methods in the superclass. Any effect that subclass methods have on private instance variables in the superclass can only result from calling public superclass methods.


    For example:
  • In the NewRefillable class that uses inheritance, none of the NewRefillable methods are allowed to use the private instance variable of Refillable, myAmount directly.
  • The myAmount variable is initialized and updated by a call to the Refillable constructor in the NewRefillable constructor using super(amount) and within NewRefillable useUp using super.useUp.
