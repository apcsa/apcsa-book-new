There are five more general areas in the course.

Inheritance. Coming next is inheritance, which we've mentioned before, in the Graphics and Listeners lessons. This is a way of writing classes that borrow from already written classes.

Arrays. You know about ArrayLists; arrays are a more commonly used but actually less powerful way of grouping things in Java. After a quick intro we'll move into working on grouping in two dimensions. 2D let's you program board games like chess, create classes that use maps and diagrams, and more. We'll also look into image and picture manipulation.

Recursion. Recursion is a simple idea—writing methods that use themselves as helpers—that you can use to write programs to solve complicated problems. We'll use recursion to draw strange shapes, solve mazes, and more.

The AP test. You'll have covered all you need to take the AP test by this point, but we'll spend some time reviewing and practicing.

A Big Project. There will generally be several weeks left in the course after the AP test. A perfect time to craft a big project! We'll offer up some ideas.
