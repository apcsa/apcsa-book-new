As you may have noted in the previous step, one criterion for comparing two sorting methods (or, for that matter, any two methods that produce the same output but in different ways) is their efficiency.

How do we measure the efficiency of a method? One way is to time it. Running times, however, will be affected by a variety of factors:

  • the computer on which it runs;
  • the Java programming environment used to compile it;
  • the number of other programs running at the same time.
