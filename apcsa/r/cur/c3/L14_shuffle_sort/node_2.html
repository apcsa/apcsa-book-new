Write a void method named perfectShuffle that simulates the way many people shuffle a deck of cards.

  • First, the cards are split into two piles, with the same number of cards in each.
  • Then, the two piles are interweaved as shown in the diagram below.

This process is called a "perfect shuffle". It is typically repeated several times. (We will shortly see that a perfect shuffle is really quite imperfect.)

In more detail, your method should perform as follows. Suppose that there are k cards in myCards. Your method should create two ArrayLists, one named topCards, consisting of elements 0 through k/2–1 of myCards, and the other named bottomCards, containing elements k/2 through k–1 of myCards. It should then interweave topCards and bottomCards into myCards as follows:

  1. element 0 from topCards
  2. element 0 from bottomCards
  3. element 1 from topCards
  4. element 1 from bottomCards
  5. etc.
