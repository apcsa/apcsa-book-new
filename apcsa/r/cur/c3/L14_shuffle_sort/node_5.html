The main part of the body of the shuffle method is a loop that runs once for each card. Each iteration finds a random card in myCards, removes that card from myCards, and adds it to the end of shuffled. After all the cards have been removed from myCards, shuffled contains the completely shuffled deck. The table below illustrates the algorithm applied to a deck of four cards.
