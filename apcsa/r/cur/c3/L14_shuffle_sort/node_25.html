Another way to sort is by counting, for each element x in the collection to be sorted, the number of elements in the collection that are less than x. The element that's less than no other—the smallest element—goes into position 0, the element that's less than just one other goes into position 1, and so on.

Here is a buggy implementation of this algorithm. 

   public void countingSort ( ) {
      ArrayList<Card> sorted = new ArrayList ( );
      for (int k=0; k<myCards.size ( ); k++) {
         int numLessThan = 0;
         for (int j=0; j<myCards.size ( ); j++) {
            if (myCards.get(k).outranks(myCards.get(j))) {
               numLessThan++;
            }
         }
         sorted.set (numLessThan, myCards.get(k));
      }
      myCards = sorted;
   }
