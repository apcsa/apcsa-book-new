All of the sorting methods we will examine use a comparison method that determines the order of two of the collection elements. In the upcoming steps, we'll be working with a deck of playing cards, so we now must define a comparison method for cards.

Provide, in the Card class, a method named outranks that returns true if either this Card's rank is greater than the argument Card's rank or the two ranks are equal and this Card's suit is higher than the argument card's suit. The order of suits is clubs (lowest), diamonds, hearts, and spades (highest).
