Here are some examples of how the ArrayList methods are used. First, we create a list of two Name references: 

   ArrayList <Name> friends = new ArrayList <String> ( );
   friends.add (new Name ("Dorothy", "Gale");
   friends.add (new Name ("Tin", "Man");

Using the two-argument add method with first argument 0, 

   friends.add (0, new Name ("Cowardly", "Lion"));

results in friends containing first the Cowardly Lion, then Dorothy Gale, then Tin Man. This has the same effect as the sequence 

   ArrayList <Name> friends = new ArrayList <Name> ( );
   friends.add (new Name ("Cowardly", "Lion");
   friends.add (new Name ("Dorothy", "Gale");
   friends.add (new Name ("Tin", "Man");

The call 

   friends.remove (1);

removes Dorothy Gale from the friends list, leaving Tin Man in position 1 and Cowardly Lion at position 0.

Finally, we can change an ArrayList item directly with set: 

   friends.set (0, new Name ("Scarecrow");

The friends list now contains Scarecrow and the Tin Man.
