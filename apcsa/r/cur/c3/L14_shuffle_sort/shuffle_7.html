Terminating an exchange-based shuffle

To start, let's look at exchange shuffling. Here is pseudocode. 

done = false;
while (!done) {
     r1 = randGen.nextInt (myCards.size());
     r2 = randGen.nextInt (myCards.size());
     Card temp = myCards.get(r1);
     myCards.set(r1, myCards.get(r2));
     myCards.set(r2, temp);
     if (we're finished) {
         break;
     }
 }

What might be a good termination condition for the loop? Try out your ideas.

It seems like we should do a lot of exchanges, so one possible way to stop the loop would be to count the iterations and stop when the count reaches 1000 or so.
Maybe the loop should continue until every random integer between 0 and the deck size minus 1 has been generated.
