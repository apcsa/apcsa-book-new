Shuffling is essentially "unsorting". It makes sense, then, that a sorting algorithm might have the same structure as a shuffling algorithm. Here is pseudocode for an algorithm named selection sort that arranges the elements of an ArrayList named myCards in order from smallest to largest. We use an auxiliary ArrayList, which we'll name sortedSoFar.

  1. Initialize sortedSoFar to empty.
  2. If myCards is nonempty, perform the sorting.
    1. Choose the smallest element in myCards.
    2. Remove it from myCards and add it to the end of sortedSoFar.
    3. Repeat this process until no more elements remain in myCards.
  3. Finally, replace myCards by sortedSoFar.
