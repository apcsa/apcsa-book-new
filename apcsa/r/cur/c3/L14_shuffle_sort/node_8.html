The methods we'll study soon use some ArrayList methods that we haven't yet encountered.

  • We have used the single-argument add method already. There is also a two-argument version. Given two arguments,

    1. an int position between 0 and size ( ), inclusive, and
    2. a reference to an object,

    add inserts the given object at the given position. Elements are shifted up if necessary to make room for the new element.

  • remove takes one int argument, and removes the item at the argument's position in the list, shifting down all the subsequent elements.

  • set takes two arguments, the first being a legal position in the list, the second a reference to store at that position.
