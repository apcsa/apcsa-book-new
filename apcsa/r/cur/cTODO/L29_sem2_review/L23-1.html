New Terms:

  • two-dimensional array

Key Concepts:

  • A two-dimensional array is an array of one-dimensional arrays.
  • A grid is a two-dimensional array with row and column indices.
  • Two-dimensional arrays are created much like one-dimensional arrays: int[][] arrayName = new int[20][30];
  • Like one-dimensional array, two-dimensional arrays can be created with contents in place: int[][] arrayName = {{1, 7, 10}, {4, 1, 3}};
  • Two-dimensional array elements are accessed by: arrayName[row index][column index];
