Linking Behaviors, States, Methods, and Variables

By now you know that behaviors and states are represented in Java objects as methods and variables. For this exercise, try representing the mouse that you just described as a simple java class. Include:

  • A class definition for your mouse
  • A list of variables needed to store the state that you described. Remember to think about what type each variable should be.
  • Method definitions for each of the behaviors that you described. Be sure to think about parameters that may be needed for these methods!
You don't need to worry about filling in your methods -- just the method definitions will do.
