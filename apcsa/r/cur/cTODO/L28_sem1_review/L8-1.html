New Terms:

  • null
  • Modulus Operator (%)
  • Helper Method

Key Concepts:

  • They keyword null represents the empty object. You can test to see if an object is null by using the == comparison operator.
  • The % (modulus) operator gives you the remainder left over from integer division. For instance, 7 % 3 = 1 because 7/3 = 2 with a remainder of 1.
  • Helper methods are methods that help other methods solve problems. They are used to make your code shorter and easier to read, in addition to improving reusability.

Your AP exam will not test you specifically on Java events. But helper methods are very important and the null keyword is fundamental to working in Java.
