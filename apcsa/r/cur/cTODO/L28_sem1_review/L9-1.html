New Terms:

  • Looping
  • loop body
  • infinite loop
  • while statement
  • counter variable
  • Debugger

Key Concepts:

  • Loops can be used to conveniently do something a large number of times.
  • A while loop lets you continue to do something a long as a condition is true.
  • Loops are often a source of bugs in programs.
  • You have to be sure that the conditions under which you end the loop will eventually be reached. Otherwise, you have an infinite loop.
  • Debuggers can be used to track down elusive bugs in your code using formalized methods.

Looping is essential to writing code that deals with complex data structures or user input. It is certain to come up on the AP exam, so you should review Lesson 9 at once!
