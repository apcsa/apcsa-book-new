New Terms:

  • Abstraction
  • Code Reuse
  • String
  • System.out.println()

Key Concepts:

  • Abstraction is the process of noticing similarities in examples and identifying the pattern that all the examples fit.
  • String objects represent sequences of characters.
  • String constants are surrounded by double-quote marks. For instance, "this could be a string constant."
  • System.out.println() is used to print messages to the Java console.

Strings have a lot of methods associated with them. Some of these are bound to turn up on the AP exam. You should go back to Lesson 3 and review them now!
