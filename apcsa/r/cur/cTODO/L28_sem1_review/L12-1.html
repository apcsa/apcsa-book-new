New Terms:

  • Collection Class
  • Index
  • ArrayList
  • for loop
  • Increment operator (++) and decrement operator (--)

Key Concepts:

  • ArrayLists are ordered collections of objects.
  • A single ArrayList can hold many objects.
  • ArrayLists have several useful built-in methods, such as add(), get(), and size().
  • ArrayLists are often processed inside of a loop so that each element can be used one at a time.
  • For loops are convenient for iterating over ArrayLists. A for loop consists of three parts: an initialization, a conditional evaluation, and an increment.
  • If the condition of a for loop is satisfied then the body of the loop is executed. Otherwise the loop ends.
  • The for-each loop automatically loops over all elements in a collection. The syntax of this type of loop is: for (ElementType variableName : collectionName){}
  • ++ and -- are used to increment and decrement a variable by 1.
