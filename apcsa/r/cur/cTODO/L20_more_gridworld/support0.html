GridWorld support classes include the following:

  • Location, representing an Actor's position in the grid;
  • Grid<E>, an interface representing grids that contain elements of type E;
  • BoundedGrid<E> and UnboundedGrid<E>, classes that implement Grid<E>
  • Actor, representing actors (Bugs etc.) in the grid

The headers for methods in these classes appear in Appendix B. The steps for this week include activities to familiarize you with these classes.
