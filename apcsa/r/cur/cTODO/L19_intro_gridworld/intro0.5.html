1. Download the GridWorld Case Study narrative and source code from the CollegeBoard website.

    1. Link to the website GridWorld Case Study College Board web site.

      • Link to the GridWorld Case Study narrative.

      • Link to the source code GridWorld Case Study source code

    2. Unzip the source code GridWorldCode.zip file. This should create a GridWorldCode folder. The source code for the projects are located in /GridWorldCode/projects.

    3. For each of the project folders in /GridWorldCode/projects, copy the contents and paste them into the corresponding project folder in the /UCCP/APCSA/Projects/lesson19 folder, which is already on your hard drive.

      For example: copy all the files in /GridWorldCode/projects/firstProject, which is just BugRunner.java, into

      Mac /UCCP/APCSA/Projects/lesson19/firstProject
      PC C:\UCCP\APCSA\Projects\lesson19\firstProject

    4. Do this for all the projects in the /GridWorldCode/projects folder.

      • firstProject
      • boxBug
      • critters

    You can now run the projects in BlueJ and start the programs by invoking the main() method of the Runner classes such as BugRunner, BoxBugRunner, etc.
