To access an element of an array, we give the name of the array, followed by the index of the desired element in square brackets. Some examples: 

monthLengths[1] = monthLengths[1] + leapDayCount (year);
for (int k=0; k<12; k++) {
   System.out.println (monthNames[k]);
}

Legal index values run from 0 through the number of elements minus 1 (the same as ArrayList index values). If the index is outside this range, an ArrayIndexOutOfBounds exception is thrown.
