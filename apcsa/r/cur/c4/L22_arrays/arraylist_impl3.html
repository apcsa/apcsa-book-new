The two-argument add method is given as its first argument the position where the new element is to be placed. Elements starting at that position must be shifted up in the array to make room for the new element. The diagram below displays the element shifting that accompanies the call 

add (2, obj);

Here, again ignoring the case where the array is already full, is the code for add. 

public void add (int position, Object obj) {
   for (int k=myElemCount; k>=position; k--) {
      myElements[k+1] = myElements[k];
   }
   myElements[position] = obj;
   myElemCount++;
}
