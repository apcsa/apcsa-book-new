ArrayLists have a size method. Arrays, on the other hand, have a public instance variable named length. The value in this variable cannot be changed. The program segments below both print the suits in suitNames. 

ArrayList<String> suitNames = new ArrayList<String> ( );
suitNames.add ("club");
suitNames.add ("diamond");
suitNames.add ("heart");
suitNames.add ("spade");
for (int k=0; k<suitNames.size(); k++) {
   System.out.println (suitNames.get (k));
}

String [ ] suitNames = {"club", "diamond", "heart", "spade"};
for (int k=0; k<suitNames.length; k++) {
   System.out.println (suitNames[k]);
}
