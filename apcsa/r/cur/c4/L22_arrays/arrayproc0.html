Patterns for array processing resemble those for ArrayList processing.

Processing every element of a values array

for (int k=0; k<values.length; k++) {
   process values[k];
}

Finding the position of an "interesting" element of values

int k;
for (k=0; k<values.length; k++) {
   if (values[k] is interesting) {
      break;
   }
}
// Either k == values.length, in which case none of the elements of values
// are interesting, or k == the position of the first interesting element.
