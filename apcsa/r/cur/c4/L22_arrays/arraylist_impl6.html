Finally, we consider the case where add is called when the myElements array is full.

We noted earlier that the number of elements an array can hold is fixed when that array is initialized. Thus we cannot expand the myElements array. What we can do is construct another, bigger array, copy the elements from myElements into it, and then replace myElements by the new array.

Here's the code:

public void add (Object obj) {
   if (myElements.length == myElemCount) {
      Object [ ] temp = new Object [2*myElemCount];
      for (int k=0; k<myElemCount; k++) {
         temp[k] = myElements[k];
      }
      myElements = temp;
   }
   // Now there is room for the new element.
   // Add it as was described earlier.
}
