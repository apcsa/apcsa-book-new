ArrayLists have two instance variables. One, which we'll call myElements, is an array that contains the list elements. (The array will typically be larger than what's needed to store the elements.) The other is an int that we'll call myElemCount; it contains the number of elements currently in the list.

The one-argument add method merely stores the new element in the first shaded spot, then increments the element count. (We'll shortly investigate what happens when the array is full.)
