The table below gives examples comparing arrays with ArrayLists.

ArrayList operations array operations
declaration ArrayList<String> suitNames; String [ ] suitNames;
initialization suitNames = new ArrayList<String> ( ); suitNames = new String[4];
declaration combined with initialization — String [ ] suitNames = {"club", "diamond", "heart", "spade"};
access suitNames.get (0) suitNames[0]
assignment suitNames.set (0, "club"); suitNames[0] = "club";

The argument to get and the first argument to set must be at least 0 and less than the number of elements added to the ArrayList minus the number of elements removed. For arrays, any index must be at least 0 and less than the number of elements set up at initialization.
