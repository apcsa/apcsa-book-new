When we discussed static methods, we briefly mentioned the special static method named main that is executed when a Java program is run without BlueJ. Its header is

public static void main (String [ ] args)

The last thing to learn about main is its argument, an array of Strings. In a system with a command-line user interface, one would run a compiled Java program with the command java followed by the name of the class whose main method is to be executed first. It is also possible in this command to provide text information to the program about to be run, by adding extra arguments to the command. These extra arguments are passed to main in the args array.

Here's an example. Suppose the main method for the Name class is run with the command

java Name jesse colin young

Then args[0] would contain "jesse", args[1] "colin", and args[2] "young" when main is entered.
