Like any Java variable, an array must be initialized as well as declared. One way of doing this is to use the new operator and give the number of elements inside square brackets. The form of such an initialization is:

arrayVar = new type [size]

Here are some examples, using the variables just declared.

monthLengths = new int [12];
stacks = new ArrayList<Card> [4];
monthNames = new String [12];

The elements of the array are automatically initialized to 0, for primitive types, or null, for references.
