Two-dimensional arrays are most often processed with nested loops. The outer loop variable is typically the row index, and the inner loop variable is the column index. The length of the array is the number of rows. The length of one of the elements of the array is the number of elements in that row.

The code below prints the values in a two-dimensional array of Strings, row by row. 

String [ ] [ ] status;
   ...
for (int row=0; row<status.length; row++) {
   for (int col=0; col<status[row].length; col++) {
      System.out.print (status[row][col]);
   }
   System.out.println (  );
}
