Not surprisingly, declaring and initializing a two-dimensional array is similar to declaring and initializing a one-dimensional array. The table below gives examples.

operation one-dimensional array two-dimensional array
declaration int [ ] values; int [ ] [ ] values;
initialization values = new int[20]; values = new int[20][30];
declaration + initialization int [ ] values = {1, 5, 7}; int [ ] [ ] values
= {{1,4,7}, {2,5,8}, {3,6,9}};
use values[k] values[row][col]
