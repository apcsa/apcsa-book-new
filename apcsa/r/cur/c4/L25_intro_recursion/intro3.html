Here again is the code for charCount. 

public static int charCount (String s) {
   if (s.equals("")) {
      return 0;
   } else {
      return 1 + charCount (s.substring(1));
   }
}

Suppose we call charCount with the argument "abc".

  • "abc" isn't equal to "", so call charCount ("bc")
    • "bc" isn't equal to "", so call charCount ("c")
      • "c" isn't equal to "", so call charCount ("")
        • "" is equal to "", so return 0
        return 1 + what the call to charCount returned, which is 1 + (0)
      return 1 + what the call to charCount returned, which is 1 + (1 + 0)
    return 1 + what the call to charCount returned, which is 1 + (1 + 1 + 0)
