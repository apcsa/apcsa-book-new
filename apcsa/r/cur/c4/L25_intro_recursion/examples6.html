A contains method can be set up with a helper method as in the sum method considered earlier. Instead of two cases—are we finished or not?— we have three:

  1. Have we looked at all the elements?
  2. If not, have we found what we're looking for,
  3. or not?

Here's the code. 

// Return true when this set contains the argument; return false otherwise.
public boolean contains (int x) {
	return containsHelper (x, 0);
}

// Return true when x is equal to one of myElements[k] through
// myElements[myElemCount-1]; return false otherwise.
private boolean containsHelper (int x, int k) {
	if (k >= myElemCount) {
		return false;
	} else if (myElements[k] == x) {
		return true;
	} else {
		return containsHelper (x, k+1);
	}
}
