A remove method is trickier. The structure is the same as that of contains, but encountering a value to be removed requires that the array be modified.

We choose to take advantage of the unordered nature of the myElements elements in implementing removal. When we encounter an element to be removed, we exchange it with myElements[myElemCount-1] (the last element of the set) and then decrement myElemCount, thereby removing it from consideration as a set element.
