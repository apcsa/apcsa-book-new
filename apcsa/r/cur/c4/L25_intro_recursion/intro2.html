Here's an example of a recursive method. 

// Find the number of characters in a nonnull 
// (though possibly empty) string.
public static int charCount (String s) {
   if (s.equals("")) {
      return 0;
   } else {
      return 1 + charCount (s.substring(1));
   }
}

Observe the two cases in this method. One, the base case, doesn't include a call to charCount. The other, the recursive case, does. The call to charCount, however, has an argument that has one fewer character than the argument passed originally to charCount. This helps ensure that execution will eventually reach the base case.
