Consider an Interval class that represents a collection of consecutive integers. For example, the interval [3,5] represents the set {3, 4, 5}. An interval whose starting value is greater than its ending value—for example, [5,3]— is empty.

A framework for this class appears below. It refers to the starting value in this interval as myLeft and the largest value as myRight, and contains a method that returns the sum of all the values in this interval.

The sum method calls sumHelper, passing it arguments that start out the recursion. This use of a top-level method to call a recursive "helper" method is typical in recursive applications. 

public class Interval {

   private int myLeft;
   private int myRight;
   
   public Interval (int left, int right) {
      myLeft = left;
      myRight = right;
   }
   
   // Return the sum of the integers represented by this interval.
   public int sum ( ) {
      return sumHelper (myLeft, myRight);
   }
   
   private int sumHelper (int left, int right) {
      if (left > right) {
         return 0;
      } else {
         return left + sumHelper (left+1, right);
      }
   }
}
