To settle the case where n and k are both 0—how many paths are there from a point to itself—we consider pathCount (0, 1). The return value for this call should be 1. In the code just designed, we treated this as a base case, but suppose we hadn't done that. We would then compute pathCount(-1,1) + pathCount(0,0). pathCount(-1,1) returns 0 because of the other base case. Thus pathCount(0,0) should be 1.
