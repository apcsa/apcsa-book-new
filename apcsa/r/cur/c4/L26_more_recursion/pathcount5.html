public static int pathCount (int n, int k) {
   if (n==0 || k==0) {
      return 1;
   } else if (n<0 || k<0) {
      return 0;
   } else {
      return pathCount(n-1,k) + pathCount(n,k-1);
   }
}

Test it with the BlueJ debugger or hand-trace it on points near (0,0) to convince yourself it works.
