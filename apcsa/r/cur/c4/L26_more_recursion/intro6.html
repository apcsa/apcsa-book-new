Consider the following version of sumHelper. As completely as possible, describe the set of intervals for which sumHelper returns the correct answer. (The answer may be "no intervals" or "all intervals".)

A good way to approach this problem is to apply the steps for understanding a recursion presented earlier, in the step titled How it works. 


private int sumHelper (int left, int right) {
 if (left == right) {
 return left;
 } else {
 int mid = (left+right)/2;
 return sumHelper (left, mid) + sumHelper (mid+1, right);
 }
}
