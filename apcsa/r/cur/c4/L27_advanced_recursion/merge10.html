Add an instance variable named comparisonCount to the merge sort program, the insertion sort program, and the selection sort program (You did those other sorts in lesson 14). Initialize it to 0 in the constructor, and then increment it for each comparison between elements of the array being sorted.

Then run all three programs using the following types of arrays:

  • an array of eight randomly chosen elements;
  • an array of eight elements arranged in decreasing order;
  • an already sorted array of eight elements;
  • an array of 256 randomly chosen elements;
  • an array of 256 elements arranged in decreasing order; and
  • an already sorted array of 256 elements.
