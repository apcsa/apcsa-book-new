Let's try it out.
First, create a new RGBImage, choosing the constructor without an argument. This should open up a dialog box for you to pick an image from what we've provided.


Pick any image you like, although I'm partial to seagull.jpg. From the BlueJ object, invoke the flipVertical method that we've provided for you. Kaboom, the seagull is flying!

If you have your own images you want to work on, put them in the images folder inside this BlueJ project, and open them the same way. Really large images might not work well!


Try clicking on the image, and you'll see a message printed out in the Java console about where you clicked and what the RGB values are there. Click on various colored regions of the image, and see if the RGB values make sense.
