Thresholding is a way of converting an image such that it contains only two possible colors for each of its pixels. Generally, these are set to be pure white and pure black. When thresholding, there is a threshold value, between 0 and 255, that controls which of the two possible values a pixel is set to.

Thresholding is used to find details in images that might otherwise be lost due to the similarity of values. With the right threshold value, those details will become much more visible.
