An image can be converted to greyscale (making it a "black and white" image, although pixels will be various shades of grey). To do this, you need to set all three color channels to the same value. However, you have to do this while maintaining the differences in brightness across the pixels.


There a many ways to do this. The easiest it to set each of the red, green, and blue values for a pixel to the average of the three.


Create a method called makeGreyscale that changes the red, green, and blue channels to make a picture "black and white".

Remember, after you are done changing the values in the objects red, green, and blue instance variables, you need to call the method refresh in order to see the changes.
